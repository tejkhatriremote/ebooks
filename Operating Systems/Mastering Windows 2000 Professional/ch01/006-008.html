TAPI Support

Telephony Application Programming Interface (TAPI) is an operating system component that offers modem access to any application that calls on it. After Windows 2000 locates and installs your modem, you never need to worry about configuring it again for the applications you use.

MAPI Support

Messaging Application Programming Interface (MAPI) is a layer of software that enables you to send e-mail from any MAPI-aware program.

Outlook Express

Outlook Express is a useful client for sending and receiving e-mail and faxes. You can use it to access LAN-based e-mail, to connect to the Internet, or to connect to online services.

Fax Services

Windows 2000 has included a fax service, which sends and receives faxes through a separate Fax Queue (just like a printer), or through various fax-aware applications like your favorite e-mail client software.

Microsoft Internet Explorer

Internet Explorer (IE) is Microsoft’s all-purpose Web browser. Much as in Windows 98, Windows 2000 has integrated IE into the desktop environment. IE functions are integrated into the entire operating system to make it easier to reach documents either on your local network or anywhere on the Internet.

Internet Information Services

Windows 2000 Professional includes the same Internet Information Services (IIS) as its big brother, Windows 2000 Server. IIS provides the ability to act as a Web server, File Transfer Protocol (FTP) server, Simple Mail Transfer Protocol (SMTP), and Post Office Protocol version 3 (POP3) server for e-mail.

DirectX Multimedia Support

DirectX is Microsoft’s architecture for accelerating the multimedia components of Windows 2000 and Windows 98. Windows 2000 supports the entire suite of DirectX 7’s components. DirectX 7 is comprised of the following components:

DirectDraw
Controls the graphics and video functions on your computer (the way that images are drawn on your screen).
DirectSound
Controls the way that sound is played on your computer.
DirectInput
Controls joystick commands on your computer.
Direct3D
Controls the manipulation of 3-D images on your screen.
DirectPlay
Provides network access for programs written to take advantage of DirectX 7.

WordPad

Like a big brother to Notepad, WordPad is a Rich Text Format (RTF) text editor, which means that in addition to just typing text into it, you can format your text and dress it up as you might with a more fully featured word processor.

Hardware Profiles

If you use your computer and peripherals in various configurations, you can use the Hardware Profiles feature to create different hardware scenarios for different environments. When the computer starts, you simply pick which profile you want to use and away you go.

Kernel-Based Graphics

Just as in NT 4, graphics functions in Windows 2000 have been moved into the Kernel, which is Windows 2000’s core software. This means significant performance increases for graphics software.

Services

Not everything that happens in Windows 2000 is noticeable even to the most expert users. Windows 2000 uses a whole set of commands, called services, that run completely in the background. Services take care of network operations, multithreaded processes, and other things that you don’t want to worry about. With Windows 2000, certain software can be installed as services in order to let them run constantly, in the background, without you ever needing to think about them. Network drivers are a common example of this type of software.

PCMCIA Support

Especially helpful for mobile users, Windows 2000 provides drivers for Personal Computer Memory Card International Association (PCMCIA) cards. What’s new in Windows 2000, however, is the support for the 32-bit CardBus interface for PCMCIA devices.

NDS Client Support

This feature enables Windows 2000 users to log on to NetWare networks that use NetWare Directory Services (NDS). Further, it allows them to continue to use all of Windows 2000’s extensive networking features within that context.

Java Support

Java is the exciting Internet programming language from Sun Microsystems that lets you run small applications—applets—over the Internet. The most innovative aspect of Java is its platform independence. That is, a Java applet doesn’t care if you are running a PC, a Unix server, or a Macintosh. Java applets use the Internet and Web browsers as their platform. Microsoft introduced Java support with IE 3. IE 5 is integrated into Windows 2000.

ActiveX Support

ActiveX is Microsoft’s programming specification for incorporating multimedia and interactive content into Internet documents such as Web pages. Like Java, ActiveX will eventually be platform independent. ActiveX was introduced with IE 3 and extended in IE 4. Remember that IE 5 of is included with Windows 2000.
