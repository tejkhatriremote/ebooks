Views

There are many ways to arrange how objects appear in Explorer, all of which are called views. The View menu not only controls how objects in the right Explorer pane appear, but also how Explorer itself can be configured to work best for you.

The view that you choose affects only the right Explorer pane. The left pane always looks the same—an expandable list of the folders and files—but you can eliminate it in Windows 2000. The default view shows the folders in the left pane, but if you press the Search button in the Toolbar you will see the Search pane on the left instead of the folders. You can also use the History button to show the recent locations on your system (or your network, or even the Internet) that you’ve visited. There are five views you can choose from to display icons in Explorer and four ways to arrange the icons after you’ve chosen your view.

The Large Icon view (see Figure 3.2) is probably the view you’re most familiar with. In this view, the icons are large and easily recognizable. Depending on how many files you have in a particular folder, the icons can take up most if not all of the space on the right Explorer pane.


Figure 3.2  This is the Large Icon view, which is Explorer’s default view.

The Small Icon view (see Figure 3.3) shrinks the size of the icons on the right Explorer pane. This makes it significantly easier to see all of the files in a folder—particularly if there are a lot of them, as in the Windows folder.


Figure 3.3  This is the Small Icon view.

The List view (see Figure 3.4) looks very much like the Small Icons view except that the objects are displayed in columns. Folders appear first, arranged alphabetically along the left side of the pane, and then the rest of the files appear, also arranged alphabetically.


Figure 3.4  This is the List view.

The Details view (see Figure 3.5) moves all the icons to the left-most column and uses the other columns for file/object information related to the icons. This information includes the file type (which is determined by how the file is associated), the size of the file, and the date the file was last modified or saved.


Figure 3.5  This is the Details view.

The newest view is the Thumbnails view (see Figure 3.6), which is particularly useful for sorting through graphic files. With this view, the graphic files are displayed as miniature versions allowing you to sort through them quickly to find just the picture you need.


Figure 3.6  This is the Thumbnails view.

So why do you care about all this organizing? Consider a situation where you can’t remember the name of a file you need, but you do remember the date it was created. Or another case where you know which application created the file and what the file extension is. With Explorer, you can arrange the items in the right pane so that they are sorted or organized by name, size, date, or file type. This makes it easier for you to better comprehend in a single glance what is and isn’t inside the folder.

To rearrange the order of your files in Explorer’s right pane, go to the View menu and click Arrange Icons. When you click there, one of those great little fly-out menus does just that (flies out, that is) and presents you with sorting options: Name and date are self-explanatory; Type refers to the three-character extension at the end of the filename (for example, this book was written as a .doc file); and size refers to the size of the file in bytes, kilobytes, or megabytes, whichever is appropriate.

You can also set the View using the Toolbar, and then right-clicking anywhere in the right pane to set the sort order. The context menu will offer choices for each of the sort orders and also give you the opportunity to tell the folder to always arrange automatically (Auto Arrange).

Toolbar

Toolbars are one of the great inventions in personal computing. They place the most common functions in a program where you can easily get to them when you want. This has been true for most Windows applications for quite a while, but it’s still relatively new to the process of file management. Sure, there was a toolbar in File Manager, but that was a little on the anemic side. Finally, Windows Explorer gives you the toolbar you’ve always wanted—see Figure 3.7.
