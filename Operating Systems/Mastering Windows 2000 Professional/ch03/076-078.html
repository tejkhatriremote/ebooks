Favorites

The Favorites menu is the same here as in Internet Explorer. Windows Explorer will use the Favorites folder as Internet Explorer, enabling you to quickly jump from browsing your local hard drive to browsing the Internet or your local intranet. There are really only two commands on the Favorites menu:

Add to Favorites
This option adds the current location (the folder you have selected in Windows Explorer, or an Internet location) to the Favorites list.
Organize Favorites
Opens a mini-Explorer to view or modify your Favorites list. The Organize Favorites mini-Explorer is shown in Figure 3.9.

Tools

The Tools menu offers commands for finding files and mapping and disconnecting network drives. The menu also has a Go To command that enables you to make a quick jump to any other folder. The tools commands are as follows:

Map Network Drive
Lets you make a new network connection. In Windows 2000, you no longer have to assign a drive letter when you map a network connection. It also lets you connect with another name or connect to an Internet server.
Disconnect Network Drive
Enables you to disconnect one or more of your current network connections.
Synchronize
Opens the Items to Synchronize dialog, and lets you set the properties of various folders on your network for offline viewing.
Folder Options
This command gives you the properties sheets for Windows Explorer. Here you can define how the Desktop will work (things like single-clicking to open programs). The View tab enables you to set properties like which files will be hidden from view. The File Types tab lets you make file associations. The final tab is Offline Files, which lets you determine which network files will be cached locally on your computer.

TIP:  The Search command lets you search for files, folders, people, and computers. It can be used to locate resources locally on your computer or across the network.

Help

As you might expect from the name, the Help menu opens Windows 2000’s Help file. Instead of just giving you the help file for Explorer as this command did in NT 4, the Help command in Windows 2000 opens the main Help file for the operating system. You can still get context-sensitive help anywhere in Windows 2000 by pressing the F1 key.

Right-Click Menus

Thanks to Windows 2000’s context-sensitive, right-click menus, you probably don’t need to use the main Explorer menus too often. In fact, the context-sensitive menus give you just as many or more options than the main menus do. By context-sensitive, we mean that the commands available on the right-click menu change depending on such factors as your current folder’s location, the current file or folder type, user privileges, and the existence of other programs installed on your computer.

Here are some of the most common commands that appear on right-click menus:

•  New (i.e., create new file or folder)
•  Cut, Copy, and Paste
•  Map Network Drive
•  Send To
•  Rename
•  Delete
•  Create Shortcut
•  Properties

Of these commands, most are equivalent to what you’d find on the application’s File or Edit menu. The Properties choice, which usually appears at the bottom of a right-click menu, generally takes you to the object’s properties sheet, a topic we discuss in more detail in Chapter 4.

Organizing Your Files and Folders

Managing your files in Windows Explorer is easy and logical. After you learn a few commands, you can use these commands over and over for a variety of tasks. After a short time using Windows Explorer, you’ll wonder how you ever did it any other way.
