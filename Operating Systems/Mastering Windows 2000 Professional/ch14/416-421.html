The final setting on the Received Faxes tab is the Send to Local E-mail Inbox option. This option lets you set the name of the e-mail profile that will receive the faxes as e-mail attachments generated by the Fax Service. This option can be set to different inboxes for each installed fax device. So, if you want to have different fax numbers for each person in your office coming into separate modems in your Windows 2000 computer, you could set each to forward all incoming faxes to that person’s inbox.

Fax Service Management Help

The next option on the Advanced Options tab of the Fax Properties dialog box will open the Help files for the Fax Service Management Console. The online Help in Windows 2000 is really outstanding when compared to other earlier operating systems. If you have questions regarding the operating of the Fax Service Management Console, this is the place to go.

Add a Fax Printer

This is the final option available on the Advanced Options tab. When you click the Add a Fax Printer button, it appears as if nothing happens. If you look in your Printers folder, however, you’ll find that another fax printer icon has been added to your installed printers. Try not to go crazy and click the button several dozen times before deciding to read the instructions. If you do, don’t worry; you can always delete all of those extra fax printers later.

The reason that Windows 2000 gives you this option is because it is sometimes helpful to have more than one configuration for a printer. The easiest way to create and manage multiple configurations for a printer is to create several icons for that printer. Then, you give each icon its own group of settings, and you’re ready to go.

Sending Faxes in Windows 2000

OK, so you’ve read through the entire configuration section and you’re ready to start faxing. So let’s get to it, then. The first thing you need to do is decide how you’re going to send a fax. There are several methods for sending a fax in Windows 2000. You can send a cover page fax through the Start menu, send a fax through your favorite word processor, or even send one by using the right-click menu, which can be accessed when you are working on documents.

Using the Send Fax Wizard

Using the Send Fax Wizard is the first method for sending faxes that we’ll look at. The wizard is opened whenever you try to send a fax outside of a program such as Microsoft Word. You can also start it by clicking Start Programs Accessories Communications Fax Send Cover Page Fax. This opens the Send Fax Wizard shown in Figure 14.9.


Figure 14.9  The Send Fax Wizard

Let’s walk through the wizard to send a sample fax.

1.  Open the Send Fax Wizard either by using the Send Cover Page Fax on the Start menu or by using the Print command in your application and printing to the Fax Printer. Click Next to begin.
2.  On the Recipient and Dialing Information page, shown in Figure 14.10, enter the recipient’s name and their fax number. If the number is in a different area code or is long distance, check the Use Dialing Rules box. When you have entered the information for one recipient, click the Add button to add that recipient to the list at the bottom of the page and clear the field so that you can type in another recipient. This lets you send the same fax to multiple recipients. Click Next when you have entered all of the recipients.


Figure 14.10  The Recipient and Dialing Information page of the Send Fax Wizard

3.  The Preparing the Cover Page dialog box asks you to define which cover page will be used for this fax. There are four cover page templates provided with Windows 2000: the Confidential, FYI, Generic, and Urgent templates. If you have created your own cover pages, they will also appear on this list (see Figure 14.11). You will also need to enter a subject line on this page before you can send the fax. Optionally, you can type a message in the Note field that will be sent on the cover page. Often, you just need to send a small bit of text information in a fax; this will enable you to do this without adding another page. Click Next when you’re ready to proceed.


Figure 14.11  The Preparing the Cover Page dialog box

4.  Next, you need to decide when to send the fax. The Scheduling Transmission page lets you choose between three time options to send the fax. The default option is Now, and sends the fax immediately. The next option is to send the fax When Discount Rates Apply. Or you can use the third option—send the fax at a Specific Time in the Next 24 Hours. If you need to enter a billing code to keep track of the faxes for accounting purposes, enter it in the appropriate field. The Scheduling Transmission page is shown in Figure 14.12.


Figure 14.12  The Scheduling Transmission page lets you control when the fax is sent.

5.  Finally, Windows 2000 will give you the Completing the Send Fax Wizard page as a summary of your selected options. This gives you one more chance to correct any mistakes you may have made. This page is shown in Figure 14.13.


Figure 14.13  The Summary page lets you verify all options before sending the fax.

When you click Next on the summary page, the fax will be sent to the Fax Queue for transmission to your recipient.

To make faxing a little easier through Explorer or My Computer, you can add a shortcut to your fax printer in the Send To menu in the right-click menu. To do this, use the following steps:

1.  Open your Printers window by clicking Start Settings Printers.
2.  Open Explorer and size the window so that you can see both Explorer and your Printers windows simultaneously.
3.  In Explorer, browse down to your Send To folder by expanding My Computer Local Disk (C:) Documents and Settings Your username Send To.
4.  In the Printers window, right-click and drag the icon for your fax printer to the Send To folder in the right pane of Explorer. Release the icon in Explorer and the context menu will pop up.
5.  Select Create Shortcut Here from the context menu. You might want to rename the shortcut something other than Shortcut to Fax Printer. It’s up to you.
6.  Close both Printers and Explorer, and you’re all done.

Now when you right-click a document, you will have an option to Send to Fax on the Context menu. When you use this method, Windows 2000 will open the Send Fax Wizard to gather the information about your recipient and the cover page you want to use, and then it will send the fax.
