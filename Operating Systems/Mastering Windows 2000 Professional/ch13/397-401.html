Like Explorer windows, the main window of Outlook Express shows folders on the left and the contents of the highlighted folder on the right. You can move a message or other item to a new folder by dragging it to the destination folder, or by highlighting it and using the Move or Copy command in the File menu or the menu that appears when you click the right mouse button.

There are several possible ways to arrange the display of files and messages in the right pane of the Windows Messaging Viewer: You can define the specific information that will appear in each column, the order in which the columns appear, and the way the Viewer sorts messages and other items.

The headings across the top of the pane identify the information in each column. Click a heading button to sort the messages according to that heading; for example, click the From heading to arrange the messages and other items by source. To change the order from ascending (earliest or smallest first) to descending (latest or largest first), use the Sort command in the View menu or right-click the column heading.

To change the set of headings in the Viewer or the order in which the headings appear, use the Columns command in the View menu to open the dialog box shown on the next page. To add a heading, choose it from the Available Columns list and click the Add button; to remove a heading, highlight it in the Show the Following Columns list and click the Remove button. The columns appear in the Viewer in the same order they appear in the list; to change the order, highlight an item and use the Move Up or Move Down button.

Searching for a Message

If you send and receive a lot of messages, you will eventually have either a bunch of folders with many messages in each one or one folder with a huge amount of stuff in it. Finding a month-old memo by looking through subject lines could take hours; fortunately, there’s an easier method. Outlook Express has a message finder that can search through all the messages in a folder or in all of your folders to find messages that match one or more criteria. Figure 13.9 shows the Find window used in the search process.


Figure 13.9  Use the Find window to search for messages.

Follow these steps to search for a message:

1.  Select the Find command from the Edit menu.
2.  Once the Find Message window shown in Figure 13.9 appears, fill in the fields that you want to use to search for messages: From, To, Subject, or Message. In each field, the results must contain your text. Use the Look In field to specify which folder to search in.
3.  Click the Find Now button to begin the search.
4.  If the Finder locates any messages that match your search criteria, it lists them in the bottom half of the Finder window.
5.  To cut off a search before it is complete, use the Stop button.

You can work with messages in the Find Message window the same way that you work with them in the main Outlook Express window. Double-click a listing to display the contents of the message. Use the commands in the menu and the toolbar to arrange columns; sort messages; use groups, filters, and views; and compose new messages.

Using Address Books

An address book is a list of names of potential recipients of Outlook Express messages. Because each address-book entry specifies the messaging service you use to reach that person, you can create a single list that includes people with whom you communicate through several services. When you want to send a message, you can choose the recipient by name and let Outlook Express worry about the transmission details.

You can open an address book from the Compose Message window by clicking the To or Cc buttons, or directly from the Outlook Express window by using the Address Book command in the Tools menu or clicking the Addresses button in the toolbar. Figure 13.10 shows the Address Book window. The Type Name or Select from List field is a smart-search tool that homes in on the name you want as you type it. In most cases, it will find the name you want by the time you type three or four letters. If you prefer, you can point directly to a name in the list and click to highlight it. Double-click an address book listing to change the properties assigned to that name.


Figure 13.10  The Address Book window helps to organize all of your messaging addresses.

To add a name to your address book, use the New Contact command in the File menu or on the toolbar. Fill in as much information as you want to track. The Properties sheets for each contact contains tabs for the person’s Name, Home, Business, Personal, Other, NetMeeting, and Digital IDs information, as shown in Figure 13.11.


Figure 13.11  The Properties sheets for a new contact provide fields for a wide variety of information.

Besides using address book listings to send messages through Outlook Express, you can also use the Personal Address Book to set up voice telephone calls through Phone Dialer. To place a call, right-click the contact you want to call, and select Action Dial to display the dialog box in Figure 13.12. Choose the number you want to call from the drop-down list and click the Dial button next to that telephone number. When the call goes through, lift the handset and start talking.


Figure 13.12  The Windows Messaging Address Book may contain multiple telephone numbers for the same person. The Phone Number drop-down box lets you access all of the numbers listed.
