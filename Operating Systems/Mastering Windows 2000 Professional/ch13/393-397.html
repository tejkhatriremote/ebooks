Configuring a New Mail Account

Whether you are creating a new identity, or using Outlook Express for the first time, you will need to configure a new account before you can send or receive e-mail. You may do this when you run the Internet Connection Wizard to get onto the Internet for the first time after installing Windows 2000, or it may be later. Either way you will be using the Internet Connection Wizard to perform the configuration.

First, the wizard will ask for a display name it will use to configure your e-mail service. This is the name that will be displayed in your e-mail message when someone reads it at the destination. Press Next when you’re satisfied. The second page of the wizard asks for your e-mail address. This defines the mailbox on the server that will be accessed. Figure 13.4 shows the second page of the wizard. If you don’t already have an e-mail address, you can click the radio button to sign up for a new mail account with Microsoft’s Hotmail service.


Figure 13.4  The Internet E-mail Address page of the Internet Connection Wizard lets you define an existing account or sign up for a new account.

The next page, the E-mail Server Names page, asks for the names of your incoming and outgoing e-mail servers. First thing to do here is tell the wizard what type of incoming mail server you connect to. The choices are POP3, IMAP, or HTTP. After you have chosen the correct setting, type the name of the server into the Incoming Mail Server text box. Then, if you have either a POP3 or IMAP server, you need to type in the name of the server that you use to send mail. The dialog box is shown in Figure 13.5.


Figure 13.5  Outlook Express needs to know the names of your incoming and outgoing mail servers.

The Internet Mail Logon page has fields to enter the username and password for your mail account. You also have the option to save your password so that you will not be prompted for it when you start Outlook Express. If your Internet mail server requires you to log on using Secure Password Authentication (SPA), check the box at the bottom of the page. When you have completed this page, click Next. The Internet Mail Logon page is shown in Figure 13.6.


Figure 13.6  The Internet Mail Logon page provides options to save your password and to log on using Secure Password Authentication.

The last page of the wizard informs you that you have successfully entered all of the information required to create the account. When you click Finish, Outlook Express will create the mail account. You can have more than one account in an identity. So it’s easy to have Outlook Express keep track of your e-mail from your home account, a club e-mail account, maybe even from work, all in one identity.

Using the Outlook Express Client

Now that you’ve set up and configured a mail account, let’s start using the client to send and receive mail. The Outlook Express client included with Windows 2000 is version 5. The new interface is shown below in Figure 13.7. It includes many new features in addition to those supplied in version 4, such as

Identities  As described earlier, identities enable you to have more than one Outlook Express user on a single computer. Identities also maintain separate Address Books.
Support for Hotmail  Outlook Express now supports sending and receiving mail through Microsoft’s free e-mail service, Hotmail.
Offline Capabilities  Outlook Express can download News and IMAP folder for offline viewing. Any replies that you make while offline will be sent when you connect again.
Contacts Pane  Outlook Express now displays the contacts that you add to your Address Book in a separate pane in the lower left of the main window.
Multiple Signatures  You can create as many signatures as you want in Outlook Express version 5, apply each one to a separate e-mail account, or choose between them whenever you feel like a change.
S/MIME Support  Outlook Express makes using S/MIME easier in version 5. It automatically looks for digital certificates on your computer and associates them with your e-mail accounts. Outlook Express also stores any digital certificate you receive with the Address Book entry for that person.


Figure 13.7  The new interface of Outlook Express.

These features, along with several others, make Outlook Express version 5 a very useful application to have in Windows 2000. The default Outlook Express window (see Figure 13.7) shows a list of messages in the current folder. You can customize the way Outlook Express organizes information in folders and the specific information that appears on your screen. Other screen layouts are discussed later in this section.

When you click the Show/Hide Folder List button in the toolbar or select the Folders command in the View menu, a display of Messaging folders appears to the left of the list of messages. This layout is similar to Windows NT Explorer’s layout. Figure 13.8 shows a folder list.


Figure 13.8  The Outlook Express folder list is similar to the list in Windows NT Explorer.

Outlook Express allows each identity to contain an unlimited number of subfolders, but at a minimum they all must include an Inbox, an Outbox, and folders for deleted items and sent items. You can use the subfolders in your identity to hold the items that you send and receive through Outlook Express, including e-mail messages, and files.
