Receiving Messages

Outlook Express checks for new mail at the same time it delivers outgoing messages to the Internet e-mail servers, which relay them to their ultimate destination. To connect to your e-mail servers, use one of the Send and Receive commands in the Tools menu.

When new messages are received, Outlook Express will notify you by playing a sound and displaying an envelope icon in the System Tray on the Taskbar.

As Outlook Express receives new messages, it stores them in the Inbox folder. It displays descriptions of unread messages in boldface type. After you’ve read a message, the description changes to normal type. To read a newly received message, open the Inbox folder and double-click the description.

Moving Messages and Files to Other Applications

Outlook Express is an OLE-compatible application. Therefore, you can drag messages and other items in the Outlook Express main window to and from other applications, including the Windows Desktop. When you drag a message from an Outlook Express folder to a drive or directory on the Desktop, Windows saves the message as a file with an .msg file extension.

You also can drag files from other applications into Outlook Express. If the file was created with an application that uses Summary Info dialog boxes, you can use the Summary fields as column headings for sorting, searching, or filtering in Outlook Express.

Summary

In this chapter, you learned how to use Outlook Express to send and receive e-mail using the POP3 and IMAP4 protocols. Outlook Express enables you to use Internet e-mail to stay in touch with friends and family members the world over. You also learned that Outlook Express can be used to maintain your contacts list, and can even integrate with Phone Dialer to call someone listed in your Contacts folder.
