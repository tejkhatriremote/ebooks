Follow these instructions to set up or change a location and its dialing properties:

1.  Open Control Panel and double-click the Phone and Modem Options icon.
2.  On the Dialing Rules tab, click the New button to create a new TAPI Location. If you are editing an existing Location, you can use the same steps except that you will be highlighting the Location you want to change and clicking the Edit button.
3.  The properties sheet for the Location opens. This is the same dialog that you can open from any TAPI-aware communications program in Windows 2000 to adjust the Location properties.
4.  Give the Location a name. In the Location Name text box, type in a name that you will use to identify this Location.
5.  Under Country/Region select the appropriate country. This will determine the correct country code for dialing long distance. In the United States the country code is 1.
6.  In the Area Code text box, type the area code for the Location you are creating.
7.  In the Dialing Rules area of the dialog you need to tell Windows 2000 how to dial from this Location. If you dial a special number to access an outside line (such as 9 on a PBX system at work), enter that in the first text box. In the second text box , you can enter the number you use to dial an outside line for long distance, if it’s different from the first number.
8.  Check the To Disable Call Waiting check box, then select the correct numeric code to disable call waiting on your phone system (if you have call waiting).

NOTE:  If your telephone line uses call waiting, click the Call Waiting option to disable it. Look in the front of your local telephone directory or call your telephone business office to find out which code you need to use, and then select that code from the drop-down menu.
9.  Finally, select the correct radio button for the type of modulation your phone system uses to dial. This will most likely be Tone dialing, but if your phone system uses Pulse dialing you will need to select Pulse in order to dial correctly.

TIP:  Insert commas into the phone number to tell your modem to pause before moving ahead to the next digit. On Hayes-compatible modems, each comma results in a two-second pause. At least one comma is necessary when you have to dial a special number to reach an outside line. So “9,” would be a typical entry in the boxes for accessing an outside line.

At this point, you may be ready to proceed with connecting to another computer. On the other hand, there is a reasonable chance that you will need to modify one or more of the rather technical settings that pertain to communicating via a modem. See the “Choosing Communications Settings” section if you want to check your settings now or if your attempt to connect fails.

Choosing Communications Settings

Unfortunately, there are no set standards covering all aspects of the technology required for modem communications. But look on the bright side: After you’ve figured out the correct settings for a given connection, HyperTerminal remembers them. From then on, all you have to do is start up the connection, and you’re almost guaranteed success.

If you are in doubt about how to set these options, there’s a simple solution. Just find out the settings for the computer at the other end of the line and set yours accordingly. If you can’t find out what those settings are, try the standard settings by dialing the connection and seeing if you can communicate. If that doesn’t work, then start systematically altering the settings one at a time, testing for a good connection each time.

To change the communications settings, choose File Properties or click the Properties button on the toolbar.

This works even if the Dial or Dialing Properties dialog box is active. You can also click Modify in the Dial dialog box. After any of these steps, you’ll see the Properties dialog box indicating the settings for the current connection, as shown in Figure 15.5.


Figure 15.5  Use the Properties dialog box for the connection to change communications settings.

NOTE:  To alter settings for a connection other than the currently open one, you must open the Properties dialog box for that connection first. You can set the Properties for any connection from the HyperTerminal folder on the Start menu (the fast way is to right-click the connection’s icon and choose Properties from the pop-up menu). Alternatively, when HyperTerminal is already running, switch to another connection by choosing File Open, then choose File Properties or click the Properties button on the toolbar.

Changes you make in the Properties dialog box apply only to the specific connection you’re working with at the time. The Connection Properties dialog has two tabs: Connect To and Settings. These tabs are specific to the HyperTerminal connection, and will potentially contain different settings for each connection you create.
