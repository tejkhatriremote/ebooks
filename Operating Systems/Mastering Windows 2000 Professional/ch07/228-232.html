.wav File Properties

The General and Preview tabs of the properties sheets of multimedia files are nearly identical for the different media types. Figure 7.11 shows an example of the Summary tab for a .wav file.


Figure 7.11  The Summary tab of a .wav file’s properties sheets

A .wav file’s Summary tab shows you copyright information (if there is any), total playback time (called Media Length), and the audio format. In this case, the .wav file has a sampling rate of 22.050KHz and a bit-depth of 16 bits, and it was encoded using the PCM codec.

TIP:  It’s unlikely that you’ll encounter .wav files using a codec you don’t have installed. If you do come across a .wav file that won’t play correctly, check its properties to see which codec it’s using. Then check in Control Panel to make sure you have that codec installed (see the section about the Devices sheet of the Multimedia Control Panel, earlier in this chapter).

.avi File Properties

In the Summary tab of an .avi file’s properties sheet, you will see more information than in the one for a .wav file, because the .avi file is using both video and audio, and both are compressed with different codecs. Like a cartoon or movie, .avi files contain video frames, and playing these frames back at a fast enough rate makes them appear as motion video. In addition to these video frames, pieces, or chunks, of audio data accompany the video. Figure 7.12 shows an example of a Summary tab for an .avi file.


Figure 7.12  The Summary tab of an .avi file’s properties sheet

Along with the copyright information, there’s information about the audio, which in this is case has a sample rate of 11.025KHz, has a bit-depth of 8 bits, uses the PCM codec, and was recorded in stereo. Next, there’s the video information, which tells us the native frame size: 320 × 240 (more on this later), total number of frames: 542, frame rate: 15 frames per second (fps), playback data rate: 118 KB/sec, and video codec used to encode: Cinepak.

TIP:  You probably won’t come across .avi files encoded using a codec you don’t have, but there are several you might see. One of these is Motion-JPEG, a cousin of the better-known MPEG codec, which is occasionally used by multimedia programs. Another one, a somewhat newer codec called TrueMotion-S, may be used in future programs. If you encounter an .avi file using a codec you don’t have installed, you will need to add it to your system using, oddly enough, the Add/Remove Hardware applet in Control Panel.

Running Windows 2000’s Multimedia Playback Applications

We’ve been spending a lot of time looking “under the hood” at Windows 2000’s multimedia subsystems, and you may be wondering, “When do we get to the good part?” Well, the reason this stuff matters is because it can make computer data come alive and let you experience this data rather than just reading characters from a screen. Developers are advancing the state of the art of multimedia and game programming at an astonishing rate. Equally astonishing is the rate at which advances in affordable multimedia accelerator hardware are being made. The overall result is smoother-running digital video, cleaner audio and more simultaneous audio channels, and more convincing-sounding music synthesis.

Running Media Player

Of all the multimedia playback applications, the most versatile is Media Player. In fact, Windows 2000 includes a new enhanced version of Media Player that is capable of playing nearly any media you want to use.

You can launch Media Player automatically by double-clicking a media file such as an .avi. Basically, Media Player will load and run whenever its services are required. You can also play .wav files with Media Player, but double-clicking a .wav file launches another application, Sound Recorder, which we get to a little later.

Using Media Player Controls

The following illustration shows the Media Player window. Certain features are common across different media types. The “transport” controls—Play, Stop, Fast Forward, and Rewind—work the same for all media types. By moving the mouse pointer slowly over a control button, you can find out what it does.

The other three Media Player controls are Skip Back, Skip Forward, and Preview Each Clip in a Playlist. The Skip Back command returns to the beginning of the selected media file. The Skip Forward moves to the end of the file. The Preview command enables you to play a short preview of each title in a playlist that you’ve assembled.

When you select Options from the View menu, you’ll see the following dialog box. Here you can customize many of the multimedia settings that were scattered all over in NT 4. For instance, you can determine how each clip will be played. Will it be played just once? Or many times? You can even select the Repeat Forever option to cause Media Player to loop endlessly. You can also set the video zoom level, which controls the default size of the window when a clip is played. The Options dialog has the following tabs:

Playback
This tab controls the options for playback of video and audio. You can set the behavior for playing clips in the Playback settings, control audio volume and balance in the Audio area, and set the zoom amount in the Video area.
Player
On the Player tab, you can set the default option for opening media files. You can use the same Media Player window to open all files, or open a separate window for each file. You can also control the display options for Media Player itself.
Custom Views
This tab enables you to customize the controls that are displayed in either the Compact or the Minimal views of Media Player.
Advanced
This is where you can fine tune the options for streaming video types such as Microsoft’s Streaming Media, or RealMedia’s RealVideo.
Formats
Formats lets you install or uninstall support for various media formats.
