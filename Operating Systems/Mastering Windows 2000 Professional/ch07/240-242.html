Recording Levels

To view recording levels, click the Options pull-down menu and select Properties. Figure 7.16 shows the Volume Control Properties dialog box.


Figure 7.16  The Volume Control Properties dialog box

In the Properties dialog box, you can select which items will be visible, and whether you want to look at Playback, Recording, or Other level settings. Select Recording for now and click OK.

You’ll notice that the mixer window’s title changes to Recording Control, as shown in Figure 7.17, and that some of the level settings are different. From here, you can adjust each one’s recording level, especially the Line In and Microphone In settings. The Recording slider on the left is effectively the “master” slider that controls all recording levels. After you’ve played with these settings a while, reset Volume Control to display playback levels.


Figure 7.17  The Recording Control window

Advanced Controls

Because we’ve got a Sound Blaster installed for our example, there’s an Advanced Controls selection under the Options pull-down menu. Enabling it places two Advanced buttons on the mixer: one under Volume Control (the master level controller) and another under Microphone. When you click either one of these buttons, you’ll see a dialog box with EQ controls for Treble and Bass. Unless you’ve got exceptionally good speakers, these controls won’t change the overall sound quality very much.

The Microphone’s Advanced dialog box also has a toggle for Automatic Gain Control, which turns down the microphone level if the incoming signal is too high.

Summary

We’ve covered a lot of technical information about numerous media data types and multimedia tools that you’ll encounter in Windows 2000. The main thing to know is that multimedia can make using Windows 2000 much more compelling, not to mention fun. Because Windows 2000 incorporates the DirectX APIs, you’ll be able to play most games written for Windows 9x on Windows 2000.

Although Windows 2000’s multimedia tools offer only basic functionality, you can still have some fun tweaking .wav files with Sound Recorder, and you can also customize your CD Player to recognize and display information about your favorite CDs. As Windows 2000’s interface suggests, explore!
