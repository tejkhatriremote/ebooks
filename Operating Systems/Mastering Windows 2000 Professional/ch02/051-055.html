Common and Private Applications

In Windows 2000, a program can be installed as a Common application or a Private application. A Common application is a program that can be used by anyone with access privileges to that specific computer. For example, if you install a copy of Microsoft Word when you are logged on to a specific workstation, that application will be available to a coworker after you log off and she logs on. Unfortunately, she could even remove the program if she so chooses.

You can see at a glance which programs and folders are Common and which are Private. The Program submenu of the Windows 2000 Start menu is separated into Private folders on top and Common folders and programs below. A computer icon on the folder represents common folders and programs.

TIP:  Folders that are marked Private are those whose access is controlled by the Windows 2000 administrator. (If you are the only person using your computer, you are that administrator, and there is no effective difference between Common folders and Personal folders, because they’re all yours.)

Installing and Removing Programs

Installing and removing programs under Windows 2000 is easy. Almost without exception, programs include their own installation utility that not only copies all their files to the necessary folders but also takes care of creating program groups and placing them on the Program submenu. Removing programs, which has become more and more difficult as programs have become more and more sophisticated and powerful, is also easy, provided you installed them using the better of the two installation procedures outlined below.

Installing Programs

There are two ways to install programs under Windows 2000. Here’s the way you’d probably expect:

1.  Insert the CD-ROM or floppy disk that contains the program you want to install.
2.  Open Explorer and navigate to the floppy or CD-ROM drive.
3.  Double-click the program’s Setup or Install program. Follow the directions that the installation routine gives you for installing the program. Many programs have numerous installation options, including Custom, which usually lets you select how many utilities and features you want to install and where you want to install them. Install these programs according to your needs and available disk space. If a program offers you a feature you don’t need, don’t install it. You can always go back later and install the program’s extra features if you change your mind.

The best way to install programs, however, is to take advantage of Windows 2000’s install utility, which launches a program’s installation routine for you while offering the added protection of monitoring the software that is being installed. There are two reasons why this monitoring process is useful. First, when a program is installed, it places files in several locations on your hard drive. These files are difficult to locate if you want to completely remove a program from your hard drive. By keeping track of what it installs onto your system, Windows 2000 makes it easier for you if you ever want to uninstall the program at some later time. Second, Windows 2000 can enforce a higher measure of security, preventing deleterious programs from causing damage while being installed.

To use the Windows 2000 installation utility, do the following:

1.  Open the Start menu.
2.  Choose Settings and then Control Panel.
3.  Double-click Add/Remove Programs.
4.  Click the Install button. You’ll see the applet shown in Figure 2.9.


Figure 2.9  Windows 2000 provides an easy-to-use installation routine.

5.  Make sure your program CD-ROM or floppy disk is in its appropriate drive, and click Next.
6.  Windows 2000 locates the installation file. Click the Finish button to complete the process of installing the program. If Windows 2000 can’t find the installation file, click the Browse button to navigate through Explorer and find the file yourself.

NOTE:  Another easy way to install an application in Windows 2000 is to use the program’s autorun feature. Most of the applications written for Windows 9x or NT 4 have this ability to automatically run the setup program when you insert the CD-ROM into the computer. This is often the easiest method to use, and usually enables you to remove the program using the Add/Remove Software applet in Control Panel.

Removing Programs

As we mentioned above, Windows 2000 can remove programs for you as well as install them.

WARNING:  Windows 2000 can only remove 32-bit “Designed for Windows 95” programs if they were installed with the Windows 2000 installation utility, or if they were specifically written to support the uninstall feature.

To remove programs, do the following:

1.  Open the Start menu.
2.  Choose Settings and then Control Panel.
3.  Double-click Add/Remove Programs. You’ll see a list of programs that Windows 2000 has kept track of and that it can remove easily (see Figure 2.10).


Figure 2.10  Windows 2000 provides a list of installed programs you can remove.

4.  Select the program on the list that you want to remove.
5.  Click the Change/Remove button to remove the program to begin the removal process.

Windows 2000 then launches the program’s own uninstall program that was included during installation. Many program suites, such as Microsoft Office or Corel PerfectOffice, let you choose whether to remove only a portion of the suite or its entirety.
