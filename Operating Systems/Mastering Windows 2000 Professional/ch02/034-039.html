Folders

In the days of Windows 3.1 and DOS, we called them directories. Now they are called folders and are more versatile than ever. Not only can they be configured to be viewed in four ways (see Figure 2.2 and the other Explorer figures in this chapter), but they also can be moved or copied anywhere you want to take them, including the Start menu and the Desktop.


Figure 2.2  Folders are easy to use and organize. Here is an icon view of folders shown in Explorer. (Compare to the list view in Figure 2.3.)

Explorer

Explorer became the shell for Windows NT 4, and it is still the shell for Windows 2000. But even though Explorer is still in Windows 2000, your desktop will look very different from NT 4. Explorer creates the views of everything you interact with, defines how things will be displayed, and provides the basic functionality of the Desktop environment. In a way, Explorer is a part of every component of Windows 2000. Mostly, Explorer is the easier to use, more capable, and more powerful replacement for File Manager. It opens a window that lists, hierarchically, the contents of everything connected to your computer: whether it be a hard drive or a printer, a device driver or the operating system’s Registry, or the hard drives or peripherals connected to another computer on your network (see Figure 2.3). You can move and copy the items listed in Explorer to initiate connections and to open or run files and applications.


Figure 2.3  Windows 2000 makes Explorer easier to use with active content to describe the contents of each folder.

My Computer

As mentioned earlier, Explorer is part of many components of Windows 2000. The tool called My Computer is no exception. My Computer is really just Explorer with a special default presentation: It displays a certain set of folders—the top-level folders for your computer’s operating system and peripherals. Double-click the My Computer icon and you have instant access to the folders for your control panels, your printers, your modem, and any storage device (hard drives, floppy drives, tape drives, CD-ROM drives, etc.) attached to your computer, both locally and via a network. The default presentation you see when you double-click My Computer uses a simple one-pane version of Explorer (see Figure 2.4). If you’d rather use the typical two-pane Explorer approach to the same information, simply right-click the My Computer icon (rather than double-click) and choose Explore.


Figure 2.4  My Computer offers a quick route to viewing the most significant components of your system—drives, printers, and control panels. This is a single-pane icon view; as with Explorer, other views are available.

You may have noticed that in Windows 2000 My Computer has changed a bit. If you’ve used Windows 98 the new look will be somewhat familiar. My Computer now shows the same single pane Explorer view as a Web page. There are some graphics added to the left side with text that shows the properties of whatever it is that you’re working with.

My Network Places

In NT 4, the Network Neighborhood was the portal to the network. In Windows 2000, My Network Places provides much of the same functionality as Network Neighborhood, but new capabilities have been added. First of all, My Network Places is a window to your local network, but it also enables you to add persistent network connections, add network printers, and yes, you can browse the network here. My Network Places uses the same Explorer view as My Computer; it supplies Web content on the left side to provide help and guidance as you use this tool (see Figure 2.5).


Figure 2.5  My Network Places is a new doorway to all the computers on your network.

Browsing the network means using an intuitive approach to click icons representing servers, hard drives, and files to find the resources you want to use.

Running Programs from the Start Menu

The Start menu is just what its name implies, the place to start working with Windows 2000. From the submenus you can access all of the various functions of your system. The default view of the Start menu will look different in Windows 2000 than in NT 4. The first thing you’ll notice is that the menus work differently. They now use a pop-out animation instead of just appearing on the screen. In addition, you can now select individual options to show or hide on the Start menu (such as the Administrative Tools group). You can still add your favorite programs to the Start menu, too.

To access a program that has been installed on your computer system, follow these steps:

1.  Click the Start button to open the Start menu.
2.  Click Programs.
3.  Choose the program group that includes the program you want to launch.
4.  From the program group’s submenu, click the program you want to launch.

NOTE:  When you install programs on your computer, you can install them into any folder or program group you like, but as a rule, programs create their own program group by default and then place that group on the Start menu’s Programs submenu.

The Keyboard Approach to the Start Menu

The Explorer interface also enables you to use the keyboard in place of the mouse. Here are some methods:

•  To open the Start menu at any time you are using NT, press Ctrl+Esc, or, if you are using a Windows 95/NT keyboard, the Windows key.
•  To navigate the menu and submenus, use your cursor movement keys as you would in any other graphical user interface, and press Enter to launch the highlighted item.
