The best-laid plans of mice and men oft gang a-gley, as the saying goes. And so, despite the tens of thousands of work hours invested in designing and coding Windows 2000 to make intelligent use of your system’s hardware, not every scenario could possibly be foreseen. Now that Windows 2000 includes the simplicity of Plug and Play, configuring and troubleshooting the system is much easier than before. Clearly, much thought has gone into reducing the perceived complexity of the operating system. Still, as we all know, human intervention can be highly advantageous at times. If you’re a systems administrator, computer jock, or power user, and you haven’t found the options you need discussed elsewhere in this book, check out this chapter—we’ve gathered all the rest of the customization and optimization options here.

Keyboard Options

Without modifying anything via the Control Panel’s Keyboard applet, your keyboard will likely work just fine. But you might want to see what you’re missing. Check out this applet’s settings (see Figure 31.1) for changing several behavioral characteristics of your keyboard— you might never go back to the defaults.


Figure 31.1  The Control Panel’s Keyboard Properties dialog box

On the Speed tab, the first setting you’ll want to fiddle with is the Repeat Delay setting. This setting determines how long you have to hold down a key before it starts repeating. If you tend to hesitate more than the average typist and you find your screen responding with tthhee instead of the, you should lengthen the repeat delay.

The Repeat Rate controls the rate at which characters appear on your screen after you press the keys on your keyboard. Most keys repeat on screen when you hold them down for a while, but this setting also controls the rate at which the spacebar, arrow keys, PgUp, and PgDn keys work as well. To alter the default setting, simply drag the slider to a faster or slower setting. You can test the effect of a given setting by clicking in the test box and holding down a letter key. If the setting is too slow or fast, adjust and try again.

At the bottom of the Speed tab, you can change the cursor blink rate. This setting determines how fast the insertion point (or I-beam) blinks when you’re editing text. Changing this to too slow a setting can cause the cursor to vanish for too long. Too fast, on the other hand, can be annoying. Drag the speed slider, then click in the test area to observe the effect.

Multiple Keyboard-Layout and Keyboard-Language Support

Because of differences in language and alphabet, computers marketed abroad have different key assignments from the standard American business computer. Key assignments in Windows 2000 for special characters such as accented characters, umlauts, and the like are determined by choices you can make on the Keyboard applet’s Input Locales tab. During installation of Windows 2000, you’re asked to confirm the language that Setup has assumed is correct for your location, so typically this will already be correct. If your line of work calls for text in several languages, you can install any number of language drivers into your system and switch them as quickly as you need them. Clicking the Taskbar or pressing a key is all you need to do to switch from one driver (and thus keyboard layout) to another.

Typically you’ll have only one driver showing when you open the Input Locales dialog. To add a new driver, click Add and choose a language from the resulting list. You may be prompted to insert a disk or CD-ROM so the applet can locate the proper driver. Then you’ll have a second listing in the Installed Languages box, shown below.

Once set to another language, some keys (shifted and unshifted) will type letters that are different from those printed on your keyboard. Unfortunately, you’ll have to experiment a bit to determine where your new keys are—Windows 2000 doesn’t include a utility that will display a keyboard map for each language driver.

If you check Enable Indicator on the Taskbar, your Taskbar will indicate which language driver is active at any one time. Clicking the indicator in the System Tray brings up a menu that lets you easily switch between languages.

If you want to become super efficient at switching between installed languages, just activate either the left Alt+Shift or Ctrl+Shift option. With one of these options enabled, simply pressing that key combination toggles between languages.

If you want to remove a language/locale driver, select it in the dialog box and click Remove. Likewise, setting the default language is as easy as clicking Set as Default. This determines which language will be used when you start up your computer.

Note that it is possible, using the Properties button, to change a locale’s keyboard layout without changing the corresponding Input Locale. In most cases, however, you will want to keep the keyboard layout the same as the Language/Locale setting.

TIP:  Are you tired of the plain-old QWERTY layout? Think it could be holding you back? If you’re willing to learn a new keyboard layout, try one of the Dvorak layouts, which are scientifically designed to increase the efficiency of your keystrokes. Normal, left-handed, and right-handed Dvorak layouts can be installed easily via the Properties button in the Keyboard Layouts list.

Hardware Settings

Finally, on the last tab, you’ll find the Hardware settings. Here you set the actual hardware driver that Windows 2000 should use with your keyboard. We say actual hardware driver to distinguish from keyboard layout or language drivers that will be produced when you’ve switched to another locale or language. Some keyboard brands require a special driver or have additional keys on them, such as number pads and the like. During Windows 2000 Setup, your keyboard was probably recognized and the correct driver installed. Should you later connect a different keyboard and it seems to be acting squirrelly after you plug it in, you should go to this tab and click the Change button. Your keyboard will then be checked for compatibility, and you’ll have the option of installing a different type of driver. Click Change and then Show All Devices to see a list of all the possible keyboard hardware drivers.
