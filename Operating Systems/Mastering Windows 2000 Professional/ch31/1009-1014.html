Regional Options

The Regional Options are for customizing Windows 2000 for use in other countries. If you’re using Windows 2000 in English in the United States, don’t bother making any changes here (unless someone has already gotten to your settings and changed them before you began working). The settings made from this box pertain exclusively to Windows and Windows applications; other programs won’t take advantage of them. Actually, even some Windows applications won’t take advantage of them. You should do some experimenting with the settings to see if they make any difference in your applications, or you should read the application’s manual for information about how to set the formats for it.

Choosing Regional Options from the Control Panel displays the Regional Options dialog box you see in Figure 31.2. The settings and their explanations follow.


Figure 31.2  The Regional Options dialog box. Changes you make here affect only the applications that use the internal Windows settings for such functions.

TIP:  Windows 2000 may ask you to insert one of the application’s disks when making changes, so have them on hand.
General
Tells Windows 2000 which country you’re in. All other settings change in accordance with Windows 2000’s understanding of the accepted practices in that country. Only bother changing the other options if you deem them necessary.
Numbers
Displays numbers with or without decimals, commas, or leading zeros, and with different decimal separators. It also determines whether to display measurements in metric or English units, and what your list separator should be. (For example, in the sentence, “Well, it’s one, two, three, what are we fighting for?”, the list separator is a comma. In other languages, items listed in a sentence are separated by other punctuation marks.)
Currency
Sets the currency indicator and the location and number of decimal digits.
Time
Allows for 12- or 24-hour time indication, A.M. or P.M. indicators, choice of separators, and leading zeros.
Date
Sets dates from a myriad of formats such as 3/6/53; 03/06/53; 3/6/1953; 06–03–1953; March 6, 1953; and others. This is useful for programs that pop the date into text at the touch of a key or translate dates from one format to another.
Input Locales
Is identical to the Keyboard Properties tab of the same name in the Keyboard applet of the Control Panel. (See the “Keyboard Options” section earlier in the chapter for a discussion of its use.) This tab lets you install additional keyboard input languages and layouts, which you can then easily switch between via a special keystroke (also configurable).

To change any of the Regional settings, simply click the appropriate tab, then the drop-down list box for the setting in question. Examples of the current settings are shown in each section, so you don’t need to change them unless they look wrong.

System Settings

There will be times in your experience with Windows 2000 Professional that you will need to adjust settings that enhance the performance of your computer. This section introduces you to the primary applets in Control Panel for this purpose. We begin by discussing the System applet and the appropriate settings and tabs available there. Then we look at the Services and Devices applets and how they can optimize the drivers that load during the system boot.

System Applet

The Control Panel’s System applet provides a number of interesting system settings. Be forewarned, however, that many of these settings are intended only for advanced users. If you happen to be familiar with the System applet in Windows 9x, you will notice that Windows 2000’s System applet is quite different—it lacks the Device Manager tab, for instance. Chances are, if you’re a power user who digs around in the computer and goofs around with boards and various upgrades, you’ll probably be using this applet from time to time. It displays information about your system’s internals—amount of RAM, type of processor, environment settings, user profiles, hardware profiles, and start-up configuration. It also lets you adjust the foreground/background multitasking performance, and the size and location of your paging file(s).

The various tabs of the System applet appear as in Figure 31.3. Let’s look at each of these in more detail.


Figure 31.3  The System applet provides advanced system settings.

General

The General tab simply displays useful information about your system, such as the version of Windows 2000 installed, the system processor type, and how much RAM is installed.

Network Identification

This tab is new to Windows 2000, and contains information formerly located under Network Properties. On the Network Identification tab, you can change your workgroup or domain membership, or the name of your computer as seen in Figure 31.4.


Figure 31.4  The Network Identification tab lets you change your computer name or domain membership.

Hardware

The Hardware tab gives you access to many of the hardware-specific settings in Windows 2000. As shown in Figure 31.5, you can access the Hardware Wizard, Device Manager, and Hardware Profiles on this tab.


Figure 31.5  The Hardware tab gives you access to most of your computer’s hardware settings.

Hardware Wizard  This wizard is useful if you are trying to add or troubleshoot hardware devices in your computer. The wizard can be used to unplug or eject hot swappable devices such as PC Cards, to install new hardware, remove hardware, and troubleshoot devices that aren’t working correctly.
