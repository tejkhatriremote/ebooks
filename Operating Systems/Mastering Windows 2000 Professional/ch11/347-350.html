Running Notepad

To run Notepad, select Notepad in the Accessories group on the Start menu. The Notepad window appears on your screen, as shown in Figure 11.1. You can immediately begin typing in the empty work area.


Figure 11.1  The Notepad window

Alternatively, you can double-click any document that Windows 2000 recognizes as a text file. As Notepad starts up, it opens that file automatically, and you’ll see the text in the work area.

TIP:  If you have Notepad files that you use regularly—for example, a file for your random notes—you might want to put them into a folder for easy access. If you use them very often, put them into the start-up folder so they are loaded when you start Windows. You can assign each one a shortcut key for rapid access.

Opening Files

After Notepad is up and running, opening another text file is as simple as choosing File Open and selecting the file you want from the Open dialog box. Of course, you can choose File New to start a new file at any time. If you’ve made changes in the previous file, Notepad gives you the expected opportunity to save it before creating the new file.

Keep in mind, however, that Notepad may not recognize the file you want to open as a text file. When you initially bring up the Open dialog box, it’s set to display only files stored with the .txt extension (note the setting in the Files of Type area). To locate a text file with another extension, choose All Files in the Files of Type area.

If you try to open a file that is too large, Notepad warns you with the message, “This file is too large for Notepad. Would you like to use WordPad to read this file?” Clicking the Yes button automatically runs WordPad, which opens the chosen file.

Notepad will go ahead and open any file you specify, even if it doesn’t contain only text. If you open a non-text file, it will probably look like unintelligible garbage.

WARNING:  Be careful about opening non-text files with Notepad. Although it’s fine to browse through a non-text file to see if you can make sense of it, don’t make any changes and, above all, don’t save the file. If you do, the file may be unusable even by the program that originally created it.

Entering and Editing Text

You can enter and edit text in Notepad as you would expect, with a few exceptions. To enter text, start typing. The insertion point moves, just as it does in WordPad. However, as you reach the end of the window, the text does not wrap. Instead, Notepad keeps adding new text to the same line, scrolling the window to the right so that the insertion point is always visible. When you press Enter, the window pans back to the far left again, ready for the next line of text. Figure 11.2 shows an example of text in this state.


Figure 11.2  Each paragraph of text stays on one long line unless Word Wrap is turned on.

NOTE:  Certain types of program files, such as .bat, .ini, and .sys files, are line-oriented and are better edited with Word Wrap off. This enables you to more clearly distinguish long lines from one another.

Working without Word Wrap is a rather inconvenient way to enter your text, because you can’t see much of what you just typed. To fix the problem, choose Format Word Wrap. After you do, the text wraps within the constraints of the window. If you resize the window, the text rewraps to fit the available space. Figure 11.3 shows the same long lines of text reformatted with Word Wrap turned on. Note that the Word Wrap setting doesn’t affect the text file itself. That is, Notepad does not insert line feeds or carriage returns at the points where the lines wrap.


Figure 11.3  Text wraps within a window if Word Wrap is turned on.

To edit your text, move the cursor to the point you want to change. You can select, cut, copy, and paste text with the mouse, using the same techniques as those described in Chapter 9. To select all the text in the file, choose Edit Select All.

Entering Special Symbols with Character Map

The Character Map program lets you choose and insert oddball characters (such as foreign alphabetic and currency symbols and characters from specialized fonts such as Symbol and Wingdings) into your documents. With Character Map, you can easily view and insert these symbols even though your keyboard does not have keys for them.

Here are some everyday examples. Suppose that instead of the standard straight quotation marks (like this: “straight”), you would prefer to use real open and close quotation marks (like this: “curly”) for a more professional-looking document. Or perhaps you regularly use the symbols for trademark (™), registered trademark (®), copyright (©); Greek letters (such as Δ); or the arrow symbols (←, →, ↑, and ↓). These, as well as fractions and foreign-language accents and the like, are included in your Windows 2000 fonts and can most likely be printed on your printer.

Character Map uses a small dialog box to display all the symbols available for each font. You select the symbol(s) you want, and Character Map puts them on the Clipboard for pasting into your document. Figure 11.4 displays some examples of special characters.
