Obtaining the Information You Need from Your ISP

Most ISPs are very good about providing exactly the pieces of information you need. In what we hope is becoming a standard, several of them are giving out handy laminated cards containing all the necessary information, complete with technical support phone numbers.

Here are the items you need to know:

•  Primary (and secondary, if it exists) Domain Name System (DNS) server
•  User ID
•  Password
•  Dial-in phone number
•  E-mail server information for sending and receiving e-mail

If your service provider does not use dynamic address assignment (most do), you also need to obtain your Internet Protocol (IP) address and gateway address from the service provider. Also, you should find out whether this is a SLIP or a PPP account (if they don’t mention this, it is likely PPP, but it won’t hurt to make sure).

After that’s done, the next step is to install your modem—by that we mean the Windows 2000 modem driver software. However, if you haven’t done so already, install the modem itself first. Installing the modem (hardware) is covered further in Chapter 27.

Installing Your Modem

If you haven’t yet installed your modem under Windows 2000, here’s how to do this:

1.  Make sure your modem is turned on (if it’s external) and is connected to your PC and phone line.
2.  Double-click the Phone and Modem Options applet on the Control Panel, and click the Modems tab.
3.  If any other modems are already installed, click Add. Otherwise, you will immediately see the Install New Modem dialog box. Click the Next button. Setup checks each serial port for connected modems. Be patient.
4.  Now, you should see a dialog box similar to Figure 16.4. If your modem was not correctly identified, click the Change button and select the correct modem (or simply a generic modem, if necessary). When ready, click Next.


Figure 16.4  Setting up a modem is made easier by Windows 2000’s capability to auto-detect most types of modems.

5.  You are now asked to enter Location Information. Change any of these settings as appropriate. If you live in an area that does not have tone dialing, be sure to change this setting to Pulse. When you are finished, click Finish.
6.  You should now be back on the main Phone and Modem Options dialog box. Note that in the future, you can easily return to this applet, highlight a modem, and click Properties (or simply double-click) to make changes to settings. For example, after you’ve verified that your modem dials and successfully connects to the Internet, you may want to set the modem speaker volume to “off” (the lowest volume setting using the slider control). When ready, click the Close button. Your modem should be ready to use!

Configuring the Dial-Up Connection

You can access Network and Dial-Up Connections either from the Start Accessories Communications menu or from the Network and Dial-Up Connections hyperlink located in the My Computer folder. Click the network and Dial-Up Connections icon, and you’ll either see a message stating that the phonebook is empty, or you’ll see the standard Network and Dial-Up Connections window. If you get the “empty” message, click OK and the Network Connection Wizard is displayed (see Figure 16.5). If you’re in the Network and Dial-Up Connections window, double-click the Make New Connection icon and this also takes you to the Network Connection Wizard.


Figure 16.5  The Network Connection Wizard helps you to make connections through Dial-Up Networking and on your local network.

The steps for completing the wizard are as follows:

1.  Start the wizard as described above to display the dialog box shown in Figure 16.5. Click Next to proceed.
2.  On the Network Connection Type page, select the type of connection you are making. For the Internet, this will typically be the second option Dial-Up to the Internet. The options are shown in Figure 16.6. Click Next when you have chosen the correct setting.


Figure 16.6  The Network Connection Wizard lets you configure many different network connections.

3.  If you have not already connected to the Internet (probably not, as that is what we’re doing, right?) the Network Connection Wizard will close and be replaced by the Internet Connection Wizard. Select the option to connect to the Internet using your phone line, and click Next.
4.  The tab shown in Figure 16.7 shows the first step of three in configuring your dial-up connection to the Internet. On this tab, you’ll need to define the phone number to dial, and optionally, the addressing information and server type. To change the server type from Point-to-Point Protocol (PPP) to Serial Line Interface Protocol (SLIP), you’ll need to click the Advanced button. If you want to assign a static IP address for your connection, or define Domain Name System (DNS) server entries, click the Advanced button, then click the Addresses tab. Most ISPs will dynamically assign these settings, so you very likely won’t need to use the Advanced settings at all. Click Next to proceed.


Figure 16.7  You will be prompted to provide the phone number information and possibly to configure Advanced options.

5.  The next step is to enter your Internet account name and password. You probably gave the ISP the name you wanted, and they probably assigned you a random password to start with. Type that information in the fields provided, then click Next.
6.  Next, give the connection a name by which to identifying it later. Windows 2000 will offer a default name of <phone number>, but you can change it to anything you like. Click Next.
7.  The final page of the Internet Connection Wizard confirms that you have successfully entered all information needed to complete the configuration. When you click Finish, the wizard will create the connection and then close.
8.  You’re now ready to try dialing in to your Internet Service Provider for the first time with this connection.
