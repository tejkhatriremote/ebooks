Web Cruising with MS Internet Explorer

Probably the first thing you’ll want to try after connecting to the Internet is cruising the Web by using your Web browser. Certainly, you can use Netscape’s Navigator browser, but because Windows 2000 comes with Microsoft’s Internet Explorer 5 (MSIE), that’s the one we’ll assume you’re using for this little test cruise.

To open Internet Explorer, double-click its icon, which you more than likely can find already installed on your Desktop. You can also see its icon on the Launch Bar, which is located just to the right of the Start button. When Internet Explorer opens, it may go to a default page or it may try to open a local page that doesn’t exist. In any case, type a Uniform Resource Locator (URL), such as microsoft.com or Webcrawler.com, into the URL/Address area and press Enter. If you’re connected, within seconds the page should begin loading.

As you jump from page to page, all previous pages you’ve visited are saved in a temporary cache, so returning to them next time is much quicker (unless the cache has been cleared in the meantime). Note that you can jump ahead and back by clicking the back and forward buttons on Internet Explorer’s toolbar. You can accomplish the same thing on the keyboard, using the Backspace key to go back a page and Shift+Backspace to go forward. Pressing the PgUp and PgDn keys scrolls up or down a screen at a time, and the up and down arrows scroll up or down by a much smaller amount. If the type is too small (or too large), click the A icon on the toolbar to cycle through the font sizes. To print whatever page(s) you’re looking at, click the Printer button on the toolbar, or select File Print. Note that font sizes when printing are affected by the current font size in the browser, so sometimes you may want to scale the font size down a bit prior to printing.

NOTE:  When you are viewing Web pages that are designed with frames, pressing the Printer button may give you a hard copy of the wrong frame. If this happens, right-click in the frame that you want printed, and select Print from the context menu.

Then, of course, there’s the Favorites button on the toolbar (and the Favorites menu), which you use to “bookmark” those places you want to return to easily later. If you’re a typical Web user, you’ll likely fill up your Favorites folder with favorite pages; at some point you’ll have to create subfolders for sorting the bookmarks into categories to make them manageable. Note that you can place these bookmarks anywhere—on your Desktop, in another folder, or even within an e-mail message (if you’re using Microsoft Outlook Express to send your e-mail, which is covered in Chapter 13, “Sending and Receiving E-Mail with Outlook Express”).

Another nice thing you can do in Internet Explorer is right-click any image in a Web page, which brings up a pop-up menu that includes options to make this image the current wallpaper or to save the image in a handy format for later wallpapering or viewing. You can also right-click any background wallpaper in the browser and set this as your Desktop wallpaper or save it to a file for later use.

The Internet Options dialog box (accessible via the Tools Internet Options menu) lets you tweak various settings, such as whether to play sounds, show video clips (or ignore them), and change the background color and default fonts. You can change the default home page, which the browser goes to when it’s first opened. On the Advanced tab, you can clear the page cache, enable use of secure protocols, enable Java Just In Time compiling, and configure the ratings system to prevent child (or adult!) access to various categories of off-color or objectionable content.

The Internet Option dialog box is also where you would go to set the address of your proxy server. If you are using Internet Explorer inside of a company with a firewall, a proxy server may be the only way to reach the Internet. A proxy translates the Web requests from computers inside the company to one “real” address on the Internet; it then passes the requests through on behalf of the client computer. If you don’t know the address or configuration of your company’s proxy, contact your network administrator.

Using Built-In Internet Utilities

Windows 2000 Professional includes several standard Internet utilities. These programs are in most cases functionally identical to the same programs or commands found on pretty much every flavor of Unix. Although most people are not going to use these nearly as often as they will their Web browsers, we describe each in the following sections so you at least know what they are. It is useful to acquaint yourself, for example, with FTP—if you are ever on a Windows 2000 system that for some reason does not have a Web browser, knowing how to use FTP to transfer files enables you to connect to Spry, Netscape, or Microsoft and download a browser.

Following this brief rundown of each of the tools, we will look at other Internet applications you can download via the Internet. Note that with each of the following programs, you can get help by typing the program name at the command prompt followed by -? (dash, question mark).
