[<InstallAdaptersSection>]  [<InstallAdaptersSection>] section is referred to by the InstallAdapters key in the [Network] section of the unattend.txt file.

•  <NetcardInfOption> = <NetcardParameterSection>

Defines the section that has descriptions for a specific network card. These descriptions can be found in the [Options] section of the card’s oemnadxx.inf file.

[<NetcardParameterSection>]  [<NetcardParameterSection>] lists the parameters for a specific network adapter card. This card should have been listed in either the <DetectAdapters> section or the <InstallAdapters> section of the unattend.txt file.

[<ProtocolsSection>]  [<ProtocolsSection>] lists the parameters for network protocols. This card should have been listed in either the <DetectAdapters> section or the <InstallAdapters> section of the unattend.txt file.

•  NBF = <NetBeuiParameters>

Implies that NetBEUI should be installed. The associated parameters section must exist to perform a successful installation.

•  NWLNKIPX = <IPXParameters>

Indicates that the IPX protocol should be installed. The associated parameters section must exist to perform a successful installation.

•  TC = <TcpipParameters>

Indicates that the TCPIP protocol should be installed. The associated parameters section must exist to perform a successful installation.

[<NetBeuiParameters>]  [<NetBeuiParameters>] should be left empty because no extra parameters are required to install NetBEUI.

[<IPXParameters>]  [<IPXParameters>] should be left empty, because no extra parameters are required to install IPX.

[<TcpipParameters>]  [<TcpipParameters>] lists the parameters required to install TCP/IP.

•  DHCP = yes | no

Use this key to tell Setup whether you want to use DHCP to set your computer’s IP address.

•  ScopeID = <scope ID>

Use this key to specify the scope if required by a network that uses NetBIOS over TCP/IP.

NOTE:  If DHCP = yes, no other keys need to be specified. If DHCP = no, then all the keys listed below must be entered in this section.
•  IPAddress = <IP address>

Used to specify the IP address of the computer.

•  Subnet = <subnet address>

Identifies the subnet mask address.

•  Gateway = <gateway address>

Indicates the default gateway address for the computer.

•  DNSServer = <IP addresses>

Identifies up to 3 DNS servers.

•  WINSPrimary = <IP address>

Indicates the IP address of the primary WINS server.

•  WINSSecondary = <IP address>

Indicates the IP address of the secondary WINS server.

•  DNSName = <DNS domain name>

Specifies the DNS domain name.

[<ServicesSection>]  [<ServicesSection>] lists the parameters required for Services installation.

•  SNMP = <SNMPParameters>

Defines the section containing SNMP parameters.

•  RAS = <RASParameters>

Defines the section containing RAS parameters.

•  NWWKSTA = <NetWareClientParameters>

Defines the section containing NetWare Client parameters.

[<NetWareClientParameters>]  [<NetWareClientParameters>] is used to specify NetWare client parameters.

•  DefaultLocation = <server location>

Indicates the default login server for the NetWare client.

•  DefaultScriptOptions = 0 | 1 | 3

Indicates the default action to be performed with scripts.

The 0 option indicates that no scripts will run. 1 implies that only NetWare 3.x scripts will run. 3 indicates that either NetWare 3.x or NetWare 4.x scripts will run.
