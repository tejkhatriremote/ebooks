Changing the Effects Settings

Think of the Effects tab of the Display properties sheets as a bonus—in Windows 95, these same features were made available via the optional Microsoft Plus! pack, in part to assist users with “legacy PCs” that did not possess the resources needed to support all the video settings. Computers running with Windows 2000 are likely to have the requisite hardware; however, if yours does not, note that two of these settings (Smooth Fonts and High-Color Icons) will not function properly if your display adapter doesn’t run in a mode that supports more than 256 colors.

In the upper part of this dialog box, you can change the appearance of the “permanent” (original default) Desktop icons by clicking Change Icon and then selecting a replacement icon from another icon stash (try using the shell32.dll for lots of icons). If you later change your mind, you can restore the default, via the Default Icon button. Figure 5.20 shows the Effects tab of the Display properties sheet. The six Visual Effects check boxes in the lower portion of the Effects tab provide additional visual enhancements to the shell. If you are running in greater than 256-color mode, try enabling all of them, and you’ll notice quite a difference. These options exemplify the usefulness of multiple settings—at lower resolutions, you won’t want them, but at higher resolutions, they can make your system look great. Explanations of each of the settings follow.

Use Transition Effects for Menus and Tooltips

This setting controls the behavior of your menus in Windows 2000. You may remember one of these effects from Windows 98 or NT 4 with Internet Explorer 4 installed. If you check the box to enable these effects, the menus will appear to be animated as they unfold.

You can select either the Fade or the Scroll effect behavior. The Scroll effect is the same as the behavior of the menus in Windows 98. The Fade effect is new. This effect causes the menus to fade in, and then fade out when you select a menu item.

Smooth Edges of Screen Fonts

Enabling this option requires video modes that support more than 256 colors. This option uses shades of gray to smooth out the jagged edges of certain letters. You will especially notice the jagged edges (or “jaggies” as some folks refer to them) if you look at letters that have rounded edges—o, p, q, s, and so on. Although font smoothing does takes care of the jaggies problem, it might make your characters appear slightly blurry or fuzzy around the edges. Try it in any case; you’ll probably like it.

Use Large Icons

If you’ve played around with different screen resolutions, you’ve probably noticed how, as the screen resolution is increased, the relative size of everything on your Desktop decreases. At 1024 or 1280 resolution, especially on a 17-inch or larger display, your icons start to look like Shrinky Dinks, and this is where Use Large Icons becomes helpful. Some applications (and the shell) supply a set of larger-sized icons, and the system makes use of these. Otherwise, the larger-sized icons are created on the fly by expanding the standard-sized ones.

Show Icons Using All Possible Colors

This option has a pleasing effect—particularly with icons that were designed using more than 16 colors. The shell contains a set of high-color icons for the folder icons, My Computer, My Network Places, Recycle Bin, and so on. As time goes on, more applications will likely ship with a set of high-color (and larger) icons built-in. As with the Smooth Edges of Screen Fonts option, you must be running in greater than 256-color mode to use this option.

Show Window Contents While Dragging

This is a fun option, provided your video controller is up to speed. In addition to letting you see the full window contents while dragging, it also gives instant feedback when resizing a window—no need to let go of the window to see how it will look when resized. This option works better (i.e., the windows slide more smoothly) if your Desktop isn’t covered by wallpaper.

Hide Keyboard Navigation Indicators Until I Use the Alt Key

This option is an interesting new addition in Windows 2000. In Windows 95 and NT 4, you would normally see the keyboard shortcuts listed alongside the menu commands in most programs. Microsoft opted for a cleaner approach, assuming that most people don’t really use the keyboard to navigate Windows 2000.

This option will turn off these keyboard shortcuts in Windows 2000 so you will see cleaner menus throughout the interface. In addition to turning off the display of keyboard shortcuts, this option also turns off the focus indicators (that little dashed line that indicates the default button in dialogs). With this option checked, these items won’t be displayed until you are actually using the Alt key to bring up a menu using the keyboard. After you start using the keyboard to navigate, the keyboard shortcuts and focus indicators will be turned on again.
