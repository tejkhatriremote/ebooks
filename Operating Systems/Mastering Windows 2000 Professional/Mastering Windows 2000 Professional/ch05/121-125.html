Setting the Background and Wallpaper

The pattern and wallpaper settings simply let you decorate the Desktop with something a little more festive than the default screen. Patterns are repetitious designs, such as the woven look of fabric. Wallpaper uses larger pictures that were created by artists with a drawing program. You can create your own patterns and wallpaper or use the ones supplied. Wallpapering can be done with a single copy of a picture placed in the center of the screen or by tiling, which gives you multiple identical pictures covering the whole screen. Some of the supplied wallpaper images cannot be used if you are low on memory. This is because the larger bitmapped images take up too much RAM.

If you are using the Active Desktop, you can use JPEG (.jpg) files as wallpaper just as you would use them for backgrounds on your Web pages. These files are great because they occupy significantly less space on the hard drive than bitmap files. Of course, with the Active Desktop you can display entire Web pages as your background if you choose. This ranges from your Home Page or some favorite Web site to custom defined pages that you specify; something that really shows what’s unique about you.

Loading a Pattern

To load a new pattern:

1.  Click the Background tab of the applet’s dialog box.
2.  Set the Wallpaper to None. Press the Pattern button. Or, if you want to use wallpaper and a pattern, make sure the Picture Display setting is Center. This setting allows the pattern to show around the borders of the wallpaper.
3.  Scroll the Pattern list to a pattern you’re interested in and highlight it. A minuscule version of the pattern shows up in the little screen in the dialog box (see Figure 5.13).


Figure 5.13  Simply highlighting a pattern displays a facsimile of it in the dialog box’s tiny monitor screen.

4.  When you’re satisfied with the pattern you’ve chosen, or if you just want to try it out, click OK to close the Pattern dialog and return to Display Properties.

NOTE:  For a pattern to show up, wallpaper must be set to None or the Picture Display option must be set to Center. This is because wallpaper always sits on top of the Desktop’s pattern.
5.  To see the effect on the whole screen, click the Apply button. This keeps the applet open and lets you easily try other patterns and settings. (If you want to leave it at that, click OK. Then the applet closes, and you are returned to the Control Panel.)

Editing a Pattern

If the supplied patterns don’t thrill you, make up your own with the built-in bitmap editor. You can either change an existing one or design your own. If you want to design your own, choose None from the Name drop-down list before you begin. Otherwise, choose a pattern you want to play with:

1.  On the Background tab of Display Properties, click the Pattern button. In the Pattern dialog, click the Edit Pattern button. If the Edit button is grayed out, you must select an existing pattern first, and then alter it to suit your taste. A new dialog box appears.
2.  In the Name text box, type a name for the new pattern, as shown in the following illustration.

3.  Create the pattern by clicking the box on the left. What you are doing is defining the smallest element of the repeated pattern (a cell). The cell is blown up in scale to make editing easier. Each click reverses the color of one pixel. The effect when the pattern is applied across a larger area and in normal size is shown in the Sample section to the right.
4.  When you like the pattern, click Add and the pattern is added to your list of patterns.
5.  Click the Done button when you’re finished creating new patterns.

If you later want to remove a pattern, select the pattern while in the editor and click Remove. If you want to edit an existing pattern, get into the editor, select an existing pattern, make changes to it, and click Change.

TIP:  If you want to abandon changes you’ve made to a pattern, click the Close button (X) in the upper-right corner, and answer No to the question about saving the changes.

Loading a New Slice of Wallpaper

The images used in wallpaper are usually .bmp files. These are bitmapped files created by programs such as Paintbrush. Other programs create .bmp files too, so the sky’s the limit as far as what you can use as wallpaper. For example, you could use a scanned color photograph of your favorite movie star, a pastoral setting, some computer art, a scanned Matisse painting, or a photo of your pet lemur. Figure 5.14 shows an example of a custom piece of wallpaper.


Figure 5.14  The Windows 2000 wallpaper

You can also edit the supplied .bmp files with Microsoft Paint if you want. Just load one into Paint by selecting File Open. In the Open dialog box, open the List Files of Type drop-down list and choose BMP file (*.bmp) to display available .bmp files. Then choose the desired file and click OK. Also note that in Microsoft Paint’s File menu there’s a choice for setting the currently open bitmapped file to Wallpaper. See Chapter 10, “Painting Pictures with Microsoft Paint,” for more information about using the Paint program.

TIP:  If you have some other form of picture file, such as a .gif or .pcx file, that you want to use, you can, but you’ll have to convert the file to .bmp format first using another graphics program such as Collage Image Manager, Publisher’s Paintbrush, or Paintshop Pro.

Here’s how to load a new .bmp file and display it as wallpaper:

1.  Create or otherwise obtain the image with whatever program you want, as long as the program saves the image as a .bmp file.
2.  Run the Display applet and click the Browse button in the Wallpaper section of the dialog box.
3.  Switch to the directory containing your .bmp file. Select the file and click OK.
4.  Click the Apply button to see the effect and keep the dialog box open, or click OK to apply the new wallpaper and close the dialog box.

Stretch Desktop Wallpaper to Fit the Screen

This option is not for everyone—or rather, every wallpaper. If you try it using tiny pieces of wallpaper (the kind intended to be tiled), you’ll probably be disappointed. However, if you happen to be using wallpaper that was intended to fill the entire screen at a lower resolution, this option does a good job of eliminating that extra border area. As the name implies, your wallpaper may actually look stretched, particularly if you’re stretching an image intended for several resolution steps below what your system is using.
