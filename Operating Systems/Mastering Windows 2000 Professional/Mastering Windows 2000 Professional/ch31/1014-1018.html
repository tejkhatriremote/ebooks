Device Manager  This tool is very much like the Device Manager in Windows 9x, except that the interface has changed to use the Management Console to display system information. Figure 31.7 shows the Device Manager.


Figure 31.7  The Device Manager lets you troubleshoot and adjust devices in your computer.

To use Device Manager to view or modify a device, expand the console tree for the type of device you want to view, and double-click the individual device. For example, if you wanted to display the properties for your video card, you would expand the Display Adapters console tree and double-click your video card’s entry. The properties sheet for a typical device lets you enable or disable a device, change the driver, or modify the resources used by that device.

Remember that Windows 2000 is a Plug-and-Play operating system, so the settings you make in Device Manager will be applied to any Plug-and-Play devices in your computer. Use this power carefully; if you make a mistake, you could potentially make your computer non-bootable.

Hardware Profiles  This applet is very handy when you’re adding or removing hardware or system drivers and want to create a backup trail. Hardware profiles are also commonly used to switch between two or more sets of connected/installed hardware devices on a regular basis. For example, they can come in handy when you need to use a laptop on the road and then connect it to a docking station at home or at the office. If you select a configuration and click the Properties button, you will see a properties sheet such as the one shown in Figure 31.8. The General tab enables you to specify the current docking status, which means you can create one hardware configuration for docked mode and another one for undocked modes. Windows 2000 then tries to automatically determine at bootup whether you’re docked or undocked, and uses the appropriate hardware configuration. If Windows 2000 cannot determine the docking status while booting, you will have to select which mode to use (or else settle for Windows 2000 choosing the Unknown Docking State mode, which might not load all the drivers you need).


Figure 31.8  The Profile Properties sheet belonging to your system’s Hardware Profiles lets you specify different docked and undocked hardware profiles.

The Network tab gives you a quick-and-easy way to disable all networking entirely via a single check box, which saves having to individually disable all your installed network drivers when you need to physically disconnect your system from the network.

User Profiles

If more than one user has logged into your system, you will find multiple profiles displayed in the profiles list on this screen (see Figure 31.9). You can delete unused, no-longer-needed profiles by simply highlighting them and clicking the Delete button. The Change Type... button lets you change a local profile into a roaming profile, and vice versa. Local profiles apply only to the one PC; roaming profiles can give you the same Desktop no matter where on the network you log in. Note that for slow-link network connections, you have the option of using a cached profile, which avoids the overhead of downloading this information from a remote server.


Figure 31.9  Each user has their own Desktop and related information, which appears when they log on to a Windows 2000 system.

The Copy To... button lets you copy a selected profile to a floppy or to some other location on the network. This gives you the ability to carry your favorite Desktop configuration with you to another system.
