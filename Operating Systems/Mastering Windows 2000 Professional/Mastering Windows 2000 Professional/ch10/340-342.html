Editing the Color Scheme

If you are working in color, you may want to create custom colors. If you are working in black and white, you may want to create a new pattern. To do either, follow these steps:

1.  Start by selecting the color you want to change in the color box. Click the chosen color to make it the foreground color.
2.  Choose Options Edit Colors or double-click the chosen color in the color box. The Color dialog box, shown in Figure 10.14, appears.


Figure 10.14  Here’s where you edit the colors that appear in the color box. In this Color dialog box, the Define Custom Colors button has already been clicked. When the dialog box first appears, you’ll see only the left half.

3.  The Basic Colors area shows more colors than are available in the color box. If you find a suitable color in the grid, click it to select it and then click OK to close the dialog box. The chosen color replaces the previously selected color in the color box.
4.  If none of the basic colors will do, click the Define Custom Colors button. The dialog box expands to show a color-editing area.
5.  In the large color-editing area, drag the crosshairs cursor to choose the desired color (it appears in the smaller area below). After you have the basic hue, you may find it easier to select the exact shade you want with the slider at the far right.
6.  When you’re satisfied, click the Add to Custom Colors button. The new color appears in the area for custom colors. When you click OK, the new color also appears as a choice in the color box.

Note that the changes you make do not affect the picture’s existing colors. A custom color set remains active only with the current picture. When you open a different picture, Paint uses the color set that was active when the picture was last saved. If you start a new picture, Paint restores the standard, or default, color set.

Saving and Loading Your Own Color Schemes

After you’ve created a custom color set, you can save it for use with other pictures by using the Options Save Colors command. And after you’ve saved a custom color set, you can use it again in other pictures with the Options Get Colors command.

Saving Your Work

If you’re painting for posterity—or at least have some use in mind for your work other than doodling—remember to save your work to disk regularly. Of course, you can open pictures you’ve worked on before for further editing (or just to admire them).

Saving Files

You can save pictures as disk files in several variations of the Windows Bitmap (.bmp) format. Normally, you can just let Paint choose the correct format for you, but there may be times when knowing which format to use comes in handy.

Here’s how the formats available in Paint work:

•  Use Monochrome Bitmap format when you have only two colors (black and white) in your picture.
•  Use 16-Color Bitmap format when you have 16 colors or fewer in your picture.
•  Use 256-Color Bitmap format when you have more than 16 and fewer than 257 colors in your picture.
•  Use 24-Bit Bitmap format when you have more than 256 colors in your picture.

Why would you want to change a picture’s format? The most common reason is that although you like a picture’s design, it looks “cartoonish” because it has too few colors. By saving the picture in a format with more colors, you’ll be able to modify it with a much richer, more realistic color palette. But this only works if your screen can display the additional colors and is set up in Control Panel to do so (see Chapter 5 for details). Note that the more colors you save, the larger the files become, and the more disk and RAM space they’ll need.

WARNING:  Saving a picture with a format that has fewer colors (for example, saving a picture with 16 colors as a monochrome bitmapped file) may ruin it. When you save a picture, Paint translates each color in the original into the closest match in the new format. Clearly, you’re likely to lose a significant amount of detail, especially when going to the monochrome format. The picture may come out looking like a sea of black with a few white dots, or vice versa.

Using Your Picture as Wallpaper

After you’ve saved a new picture, or if you’re working with a picture you opened from disk, Paint can tell Windows 2000 to use the picture as wallpaper. From then on, the picture will appear as the backdrop for your Desktop. Paint handles the details for you; you don’t need to use the Control Panel at all. (See Chapter 5 for more on wallpaper.)

Here’s how to set the current picture as wallpaper:

1.  Save the picture to disk if you haven’t already done so.
2.  Choose File Set as Wallpaper (Tiled) if you want multiple side-by-side copies of the image on your Desktop. Choose File Set as Wallpaper (Centered) if you want a single copy of the image centered on the Desktop.

Summary

In this chapter, you have learned to work with the Paint application to create and modify pictures in Windows 2000. Whether you are capturing a screen shot and editing it for size and quality, or creating your own refrigerator art to compete with your kids, Paint can handle the job.

You also learned how to apply more advanced techniques, such as cutting portions of an image and using them to create custom brushes for painting. Finally, we discussed how to save your picture and display it as wallpaper in Windows 2000.
