Running Programs Minimized

Every program on your hard drive can be set to run minimized. When you minimize a program, you shrink the size of the program window so that it appears only as a button on your Taskbar. By setting the Run Minimized command in an application’s Properties page, every time you start that program it automatically runs minimized.

Running a program in a minimized window can be helpful if you want to start a program that you know you will need to use, but not immediately. Or, if you start certain programs automatically, you can have them run in a minimized window and not clutter your Desktop right away.

To set an application to run minimized, select its icon, right-click it, and choose Properties. Click the box labeled Run Minimized and click OK. You can also select Properties from Explorer’s File menu, or you can simultaneously press the Ctrl and the Enter key on your keyboard.

Special Topic: Running DOS Applications

There are two ways to run DOS applications under Windows 2000. You may ask, “Why would I want to bother with DOS, anyway?” Well, it’s not always a question of wanting to; sometimes it’s a question of needing to run DOS. There is a tremendous installed base of DOS software, and you never know when you might have to run some of it in order to finish a project started by someone else or to look at something historical in its original application. (Or maybe you just can’t get enough of your favorite DOS games. That’s a necessity, right?) Windows 2000 provides not one, but two, ways to run DOS applications.

NOTE:  You don’t need to have DOS installed on your system to run a DOS application under Windows 2000.

The easiest way to run a DOS program is to run it the same way you would run a Windows program: Locate the DOS program by using Explorer, and then double-click its icon to launch it. No problem.

Many computer users are also comfortable using the command prompt (sometimes called the DOS prompt, even though in Windows 2000 it isn’t DOS) to give instructions to their computer, including instructions to run programs. In some cases, using the command prompt is faster because you avoid the overhead used by the Windows graphical user interface (GUI).

To use the command prompt to run a DOS program, click the Start menu; then, on the Programs submenu, click Command Prompt. This will open a window that looks remarkably like DOS. This is actually the 32-bit command line interface of Windows 2000. From this prompt you can type the path and name of the DOS program you want to run, for example, C:\wp5\wp51, to run WordPerfect version 5.1.

When you use the command prompt, most of the old DOS commands you may remember will work. The exceptions, as mentioned previously, are any commands that try to access hardware directly, as outlined below:

•  You can’t use SCANDISK (formerly used to check hard disk errors). Instead, use the error-checking tool available through the drive’s properties sheet to check for disk errors or run CHKDSK/F from the command prompt.
•  You can’t use DEFRAG (formerly used to defragment your hard disk). Instead, use the new Disk Defragmenter that is located in the System Tools folder on the Start menu.
•  You can’t use LOCK (formerly used to reserve direct access to the hard drive). Instead, you can use Windows 2000’s workstation administration features to grant and verify rights to specific users and objects.
•  You can’t use any DOS commands that compromise security.

Switching from One Window to Another

Windows 2000 provides some easy ways to go back and forth between your open windows and applications.

Using Alt+Tab to Switch

The keyboard approach to switching from one task to another is the Alt+Tab key combination. When you press these two keys together, a dialog box opens, showing icons and names for each of the items you have open. (Actually, this was available in earlier versions of Windows as well, but from many reports the technique wasn’t well known.) Hold down the Alt key while repeatedly pressing the Tab key to highlight each icon in a cycle. When the item you want to go to is highlighted, release the Alt+Tab keys to go to that item’s window.

Using the Taskbar to Switch

One of the biggest criticisms Microsoft received regarding the Program Manager approach to Windows computing was that you couldn’t tell at a glance which programs you were running at any given time. Their solution to this problem is the Taskbar. Everything you have opened, including windows for folders and drives, is listed on the Taskbar.

The Taskbar isn’t just for display, however. Each smaller bar you see on the Taskbar (shown in the following graphic) acts as a button. You can click the button to move the Desktop’s focus to the item it represents, thus bringing its window to the front if it has been obscured, or maximizing it if it has been minimized.
