Tape Drives

Even though tape drives can also be SCSI drives, they configure differently than SCSI hard drives and CD-ROMs. However, thanks to SCSI, this particular driver is probably the easiest driver to install. To add a tape drive to your system, do the following:

1.  Make sure that the SCSI address for the tape drive is set to a number that is not being used by another SCSI device. SCSI addresses range from 0 to 7 for SCSI-2, and 0 to 15 for SCSI-3. For example, say one user is adding an APS DAT drive to his workstation. On the back of the drive is a switch clearly labeled “SCSI address” that enables him to change the address easily. When purchased, the switch was set to 6. That address is already being used by a SCSI hard drive, however. So, he simply flips the switch to 5. If he did not change the setting, upon start-up the two SCSI drives would conflict and the computer wouldn’t operate at all. The only remedy would be to remove the tape drive or change the SCSI address so that the computer could start properly.
2.  After you have set the address, connect the drive into the SCSI port on your computer.
3.  Start the computer. If Windows 2000 doesn’t automatically find your tape drive during the startup, open Control Panel and run Add/Remove Hardware to detect the drive.
4.  After the tape drive has been detected, you need to install the driver. Click OK and Windows 2000 loads the driver for you. Device Manager (see Figure 27.4) lists the tape drive that has been detected and confirms that its driver has been loaded.

Modems

Modems are essential to make use of any remote services in Windows 2000. Connecting your workstation to the network from a remote location is practically impossible if you don’t have a modem.

Installing a New Modem

Installing a modem is fairly simple and straightforward, as you will see. You can use Windows 2000’s Install New Modem Wizard to add a new modem to your computer, or use the Add/Remove Hardware Wizard.

Follow these steps to install a new modem in the system:

1.  Open the Control Panel and double-click the Phone & Modem Options icon. Click the Modems tab and click the Add button. The first page of the Install New Modem Wizard dialog box appears, as shown in Figure 27.5.


Figure 27.5  The opening dialog box of the Install New Modem Wizard

You are given the choice of selecting your modem from a list or letting Windows 2000 automatically detect your modem. You might want to pick the modem yourself just in case Windows 2000 can’t detect your modem, which happens occasionally.

2.  Check the box to select your modem from the list and click Next. Select your modem from the list. If your modem isn’t on the list, you have the option of loading the driver from a disk. The modem’s manufacturer provides this disk. Click the Have Disk button to add the driver. Otherwise, click Next.
3.  Choose the COM port that you wish to use and click Next. You should receive a message that confirms the installation.
4.  Click OK.
5.  If this is the first modem installed in your computer, you may be prompted to provide some information. Supply the necessary information regarding the location of your computer in the Location Information dialog box (see Figure 27.6). Select the appropriate country and enter the area code from which you are calling. If your phone system requires that you dial a number to access an outside line, enter that as well. Also, indicate whether your phone system uses tone or pulse dialing.


Figure 27.6  Location Information dialog box

6.  Click Next to finish setting up the modem.

Changing Modem Settings

If you choose to change the settings of your modem, you’ll have to take different measures. After a modem has been installed, you’ll use the Modems tab of the Phone & Modem Options applet (shown in Figure 27.7) to make changes to all modem settings.


Figure 27.7  Modems tab of Phone & Modems Options

Modem Properties

Highlight the modem that you want to change and click the Properties button. The screen shown in Figure 27.8 is where you can change the speaker volume and the maximum speed of the modem.


Figure 27.8  Setting the modem properties

The Diagnostics tab is new in Windows 2000 (Figure 27.9) but you may be familiar with it from Windows 9x. This tab enables you to query the modem to see if it is installed correctly and can respond to simple AT commands.


Figure 27.9  The Diagnostics tab lets you test your modem.

Click the Advanced tab to change other settings such as error control, flow control, and modulation type. The only setting right on the tab is Extra Initialization Commands, which will be appended to the modem’s initialization string. Figure 27.10 shows the Advanced tab. The other buttons on the Advanced tab provide access to the Advanced Port Settings and the default dialing properties.


Figure 27.10  Advanced tab of the modem Properties

Dialing Properties

Click the Dialing Rules tab in Phone & Modem Options (see Figure 27.11). Now you can add a location (in case you’re using a portable computer) and change the area code and country. You can also change the number used to dial an outside line or long-distance phone number, disable call-waiting, indicate tone or pulse dialing, and indicate whether you will be using a calling card.


Figure 27.11  The Dialing Rules tab

To make these changes to an existing location, simply double-click the entry in the Locations list that you want to modify. To create a new Location, click the New button and fill in the required information.

If you do decide to use a calling card when dialing, you need to indicate the type of card you are using and the card number. Click the Calling Card tab, and then choose the card you’re using from the list. You will need to enter your calling card number and your Personal Identification Number (PIN).

If your card is not on the list, you’ll need to provide the name of the card by clicking New and typing the name of the new card. You’ll also have to provide the rules required to use the card. If you’re using a card that’s already on the list, click Edit to see what dialing rules are in effect for your card. The Calling Card tab in Figure 27.12 shows an example of the rules for an AT&T calling card when dialing 1-0-1-0-ATT-0.
