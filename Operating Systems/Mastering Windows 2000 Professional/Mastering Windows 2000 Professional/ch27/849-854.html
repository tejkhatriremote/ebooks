The process of installing new hardware in Windows 2000 has changed radically from the procedure used in NT 4. For one thing, you now have Plug and Play to assist you in locating and configuring new hardware. The other important thing to note is that you can locate all of your hardware configuration information in Device Manager. Nonetheless, this is a new operating system, and things don’t work the way they did in Windows 9x, so we’ll explain how to install new hardware using these new tools in Windows 2000.

TIP:  Only users with domain administrator or local administrator permissions are allowed to add or change hardware on a workstation.

Display Adapters

Generally, the only time you’ll need to install a new display adapter is when you add a new video card to your system. Just in case you do muck up your video driver, however, you can always reinstall it—that is, if you can still see what you’re doing. Remember the Last Known Good configuration that we mentioned in Chapter 20, “Installing Windows 2000 (and Automating Installation)”? It won’t help if you’ve already tried to log on. If, however, you still have the VGA mode option available to you when loading Windows 2000, you’ll be able to boot the machine and detect the new display adapter.

Installing the driver is easy. Follow these steps:

1.  Shut down your computer and turn off the power. Install the new display adapter according to the manufacturer’s instructions.
2.  Reboot the computer and when you see the menu options to start Windows 2000, press the F8 key. From the F8 menu, select Safe Mode VGA to start the computer with a standard VGA driver.
3.  Once Windows 2000 has started and you have logged on to the system, open Add/Remove Hardware Wizard, shown in Figure 27.1.


Figure 27.1  The Add/Remove Hardware Wizard makes it easy to detect and configure new hardware.

4.  Click Next, and select the option to Add/Troubleshoot a device. Click Next again, and select Add A New Device from the listed options. Click Next to display the dialog box shown in Figure 27.2.


Figure 27.2  The wizard gives you the option of searching for new hardware or selecting it from a list.

5.  Click Yes, Search For New Hardware and click the Next button to begin the search. If things go well at this point, Windows 2000 will correctly identify your new display adapter and remove the settings for your old adapter.
6.  Click Finish to allow Windows 2000 to complete the installation.

Here are a couple of additional points concerning display adapters:

•  When troubleshooting your video platform, remember that VGA is a standard. If your video card is behaving poorly, first move your resolution back to 640 × 480. If this setting does not resolve your problem, reinstall your video driver in VGA mode.
•  When choosing to use a Super VGA card, try to find one that supports the S3 standard—it seems to be a de facto standard in the industry.
•  Visit the manufacturer’s Web site to determine if they have Windows 2000 drivers for your card. This might actually become a part of your hardware selection process.

SCSI Cards and Devices

In this section we focus on how to install three SCSI devices: SCSI hard drives, CD-ROMs, and tape drives. The primary fact to keep in mind is that purchasing a device that simply has the SCSI designation is no guarantee that it will work with other SCSI devices. Always look for the SCSI-2 or SCSI-3 designation, which ensures that your SCSI host adapter and SCSI devices will interact with each other. Also check to make sure that the SCSI devices you are installing are on the Hardware Compatibility List (HCL) for Windows 2000. You can find the HCL either in the package with Windows 2000 Professional, or on Microsoft’s Web site at: http://www.microsoft.com/isapi/hwtest/hcl.idc.

Hard Drives and CD-ROMs

Although hard drives and CD-ROMs are usually part of the initial installation of Windows 2000 Professional, you may choose to add a hard drive or CD-ROM after Windows 2000 has been installed. If so, the basic approach will be the same as when we added a display adapter above. In fact, this approach is the same no matter what type of new hardware you are installing. To display your current settings, right-click My Computer and then choose Properties. On the Hardware tab, click the Device Manager button to display the console shown in Figure 27.3.


Figure 27.3  Device Manager will display any installed SCSI components.
