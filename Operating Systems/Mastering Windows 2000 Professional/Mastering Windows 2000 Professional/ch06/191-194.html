Printing from a Program

Unless you choose otherwise, the output from Windows and DOS programs are shunted to the Print Manager for printing. If no particular printer has been chosen (perhaps because the program—for example, a DOS program or Notepad—doesn’t give you a choice), the default printer is used. You can set the default printer at any time by right-clicking a printer icon and choosing Set as Default.

The appearance of your printed documents varies somewhat from program to program because not all programs can take full advantage of the capabilities of your printer and printer driver. For example, simple word processing programs such as Notepad don’t let you change the font; a full-blown word processing program such as Word Pro can print all kinds of fancy graphics, fonts, columns of text, and so forth.

TIP:  Here’s something to consider if you are still using DOS programs: If you choose as your default a printer that your DOS programs can’t work with, your print jobs could bomb. Suppose you’re running WordPerfect 5.1 for DOS and have it installed to print to an HP LaserJet, but the default printer is a PostScript printer. Your printouts would be nothing but a list of PostScript commands, such as ERROR: undefined, OFFENDING COMMAND: |ume STACK {—pop—, and so on. On another type of printer, you might see garbage printouts, such as @#$@$%G$V%B^YB&M(OM instead of readable text. Just make sure your DOS programs can work with your default printer. Also, depending on how the DOS program accesses the printer, it might not work at all under Windows 2000.

When you print from any program, the file is printed to a disk file instead of directly to the printer. Print Manager then spools the file to the assigned printer(s), coordinating the flow of data and keeping you informed of the progress. Jobs are queued up and listed in the Print Manager window, from which their status can be observed; they can be rearranged, deleted, and so forth.

To print from any program, including Windows (both 16-bit and 32-bit) programs, follow these steps:

1.  Check to see that the printer and page settings are correct. Some program’s File menus provide a Printer Setup, Page Setup, or other option for this. Note that settings you make from such a box temporarily (sometimes permanently, depending on the program) override settings made from the Printer Properties dialog box.
2.  Select the Print command on the program’s File menu and fill in whatever information is asked of you. For example, in WordPad, the Print dialog box looks like Figure 6.19. Some programs have rather elaborate dialog boxes for choosing which printer you want to print to, scaling or graphically altering the printout, and even adjusting the properties of the printer. Still, you can usually make only the most obvious settings, such as the following, and get away with it:

•  Correct printer
•  Correct number of copies
•  Correct print range (pages, spreadsheet cells, portion of graphic, and so on)
•  For color printers, which ink cartridge you have in (black-and-white or color)

Printing by Dragging Files

You can quickly print Windows program document files by dragging them onto a printer’s icon or window. You can drag from the Desktop, a folder, the Find box, Explorer, or the File Manager window into the running Print Manager. This works only with documents that have an association with a particular program. (See Chapter 3, “Organizing Programs and Documents,” for a discussion of associating.) To check if a document has an association, right-click it. If the menu that pops up has an Open command on it (not Open With), it has an association.

To use this technique, first arrange your screen so you can see the file(s) you want to print as well as either the printer’s icon or its window. (You open a printer’s window by double-clicking its icon.) Then drag the document file(s) into the Print Manager icon or window. The file is loaded into the source program, the Print command is automatically executed, and the file is spooled to Print Manager. The document isn’t moved out of its home folder; it’s just printed.

TIP:  The drag-and-drop method can be used with shortcuts, too. You can drag shortcuts of documents to a printer or even to a shortcut of a printer, and the process will still work. If you like this way of printing, keep a shortcut of your printer on the Desktop so you can drag documents to it without needing to open the Printers folder. Double-clicking a shortcut provides an easy means of checking its print queue, too.

Printing by Right-Clicking a Document

Another printing shortcut you can use with some documents doesn’t even require you to have a printer icon in view. Right-click the icon of any document you want to print.
