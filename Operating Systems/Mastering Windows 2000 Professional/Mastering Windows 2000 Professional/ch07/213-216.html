Installing the Sound Board in Your PC

Windows 2000 Professional ships with drivers on its installation CD-ROM for the most popular sound cards out there. For our example, let’s install a Sound Blaster 16 sound card with a Roland SCD-10 General MIDI daughterboard to make it interesting (this will help show off Windows 2000’s MIDI functionality later).

The first step is to check which I/O resources the Sound Blaster needs. It turns out that the Sound Blaster requires two base I/O addresses, one IRQ, and two DMA channels. Its default settings are

•  Base I/O addresses: 220 and 330
•  IRQ: 5
•  DMA channels: 1 and 5

Checking our table of I/O resources, we see that the second base I/O address will conflict with our network card, so we need to use another address. Sound Blaster can use either 330 or 300 for its MPU-401/General MIDI, and because no devices are using 300, we select that address. Looking at the rest of the requirements, we see that there are no other I/O conflicts, so we’re ready to install the board into the PC. Be sure to add the I/O resources you’re using to your table of installed devices, because you’ll need to tell Windows 2000 what they are.

WARNING:  Make sure that your PC is turned off, and that you first touch your PC’s metal chassis to discharge any static electricity before adding or removing any expansion boards!

Installing the Sound Card Driver

After you have installed the sound board in your PC and replaced the PC’s cover, restart the system and follow these steps to install the sound card driver in Windows 2000:

1.  Open Control Panel and double-click the Add/Remove Hardware icon. This will start the Add/Remove Hardware Wizard that will guide you through the process of adding the card. Click Next.
2.  On the Choose a Hardware Task dialog, select Add/Troubleshoot a Device, then click Next (see Figure 7.2).


Figure 7.2  The Add/Remove Hardware Wizard also lets you troubleshoot installed devices.

3.  The New hardware Detection dialog opens, and Windows 2000 begins searching for the newly installed sound card. At this point, Windows 2000 will only be looking for Plug-and-Play devices that can be installed and configured without user intervention.
4.  If the device was detected correctly, Windows 2000 will install the driver. If nothing was detected, you will be presented with the Choose a Hardware Device dialog, which allows you to select the type of device from a list (see Figure 7.3). In this list, select the first option, Add a New Device, then click Next.


Figure 7.3  You can choose a device to troubleshoot or install a new device using the Add/Remove Hardware Wizard.

5.  On the Find New Hardware dialog, select the option you want. If you want Windows 2000 to search for the device click Yes, Search for New Hardware. If you know the information for the device and do not want to search, or you have drivers from the manufacturer, select No, I Want to Select the Hardware From a List.
6.  Windows 2000 will search for any non-Plug-and-Play hardware installed in the computer and, if possible, install a driver for it. If the hardware is not detected, press Next to pick the device from a list.
7.  On the Hardware Type dialog, select the type of hardware you are installing. For a sound card, select Sound, Video, and Game Controllers. Click Next.
8.  Select the manufacturer and model of the device you are installing (see Figure 7.4). If it isn’t listed you have the option of clicking the Have Disk button to specify the location of the updated driver. Click Next.


Figure 7.4  When manually defining a new device, you have to select the manufacturer and the model of the device.

9.  Windows 2000 will give you a chance to change your mind by displaying the choice you’ve made. It also informs you of the settings that it will use to install the device, typically the default settings for the device.
10.  Windows 2000 will copy the files for your device, then display a summary of the installation. If you need to make settings for the card, a dialog will be displayed asking for the correct settings. The default options listed will often be correct, but it’s a good idea to check them anyway. Click OK to accept the settings. Click Finish to complete the installation process and close the wizard.
11.  You may or may not need to reboot the computer before the sound card will work. Either way, plug in some speakers or headphones to hear some sound. Be very careful with headphones, though. The volume of the mixer is often quite loud when the sound card is first installed.

WARNING:  Be careful using headphones with a sound card. When first testing your sound card after driver installation, hold the headphones near your ear, but don’t cover your ears with them. Windows 2000’s start-up sound can be painfully loud, so adjust your mixer settings before wearing headphones.

After you reboot and log on to Windows 2000, you should hear the Windows Logon Sound, which is the Windows default start-up sound. If you don’t hear this sound, read the next section. If you heard the Windows Logon Sound, this indicates that the wave audio portion of your sound card is working fine. To test the General MIDI portion of the sound card, follow these steps:

1.  Go to the C:\winnt\media subdirectory, and find a file called canyon.mid.
2.  Double-click the canyon.mid file to launch Media Player. (Media Player is covered later, in this chapter’s “Running Media Player” section.)
3.  If the music doesn’t start automatically, click the Play button in Media Player, which is the one furthest to the left. You should hear a song—in this case a MIDI sequence.
