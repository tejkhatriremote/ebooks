Editing CD Player Information

CD Player enables you to enter album and track title information for discs that aren’t located on the Internet. This way, the album information is displayed the next time you run CD Player just like the albums that are automatically downloaded from the Net. It’s also useful if your favorite album isn’t in the databases on either of the Internet sites. You can enter the correct information and upload it to the sites so they can add it to their database.

1.  Insert an audio CD into your CD-ROM drive.
2.  Click the Options button to drop down the menu and select Playlist. Highlight the disc that’s currently in the drive and click Edit Playlist to display the dialog box shown in Figure 7.14.


Figure 7.14  Using CD Player’s Playlist, you can edit disc and track names, as well as specify play orders.

3.  To change the Artist name, highlight the text in that field, and begin typing.
4.  Press the Tab key and enter the CD title.
5.  For each of the tracks, you can either highlight a given track in the Playlist box or in the Available Tracks box, and enter a name in the field in the middle of the dialog box, which displays the track number.
6.  When you’ve finished, click the Add to Playlist button. You’ll see that the name is updated in both the Available Tracks and Playlist boxes.
7.  After you’ve entered all the track names, click OK.
8.  As a test, remove the CD from the drive, and then put it back in. CD Player reads and recognizes the disc, and shows you the information you’ve just input.

Changing Time Display Options

The four CD Player time display options are straightforward:

Track Time Elapsed
Shows how far you are in playing a track.
Track Time Remaining
Displays the amount of time left for a given track.
CD Time Elapsed
Shows the total time that the disc has been playing.
CD Time Remaining
Shows the total amount of play time left for the entire disc.

You can change the time display mode by clicking the Options button and selecting Preferences from the menu. On the Player Options sheet, select the mode you want to use. Or simpler still click the display itself, and it cycles through the four display modes.

Making CD Player Preferences

The last thing to check out is the CD Player’s Preferences dialog box. To get there, click Options and select Preferences. Here you can toggle several features:

•  Whether CD Player begins playing when you open it
•  Whether CD playback stops when you exit CD Player
•  Whether you want CD Player to always be the top window
•  Whether you want to display the control on the Taskbar

You can also adjust the Preview Time interval, which defaults to five seconds. Finally, you can press the Advanced Audio button to set the default CD-ROM drive for audio CDs, select an audio mixer, and select a default volume control.

Running Sound Recorder

Windows 2000’s Sound Recorder offers basic wave editing functionality. The keyword is basic. Other full-featured shareware and commercially available wave editors offer much more complete and precise editing tools for .wav files. Sound Recorder suffices for simple edits, but if you really want to tweak .wav files, you may want to consider another editor.

The first thing to know is that Sound Recorder is a destructive editor. If that sounds a little ominous, that’s because there’s no Undo feature in Sound Recorder. In other words, if you make a change, and then decide you don’t like the result, the only way to undo it is to either exit the application without saving or open a new or existing .wav file, without saving the one you’ve just been editing. The best approach is to make a copy of the file you want to edit, and do your destructive edits on the copy. Then, if you’re not pleased with the edits, the original is still intact.

To launch the Sound Recorder, click the Start button, choose Programs, then Accessories, then Entertainment, and then Sound Recorder. The first things you see are the transport controls: Rewind, Fast Forward, Play, Stop, and Record. The Sound Recorder window looks like this:

Playing Back a .wav File

To get an idea of how the Sound Recorder works, let’s make a copy of the tada.wav file located in the C:\Winnt\Media subdirectory:

1.  Open your hard drive’s window.
2.  Go into the Windows subdirectory. Find the tada.wav file toward the bottom of the file listing.
3.  Hold down the Ctrl key and drag the file onto your Desktop. You’ll see a small plus sign, which indicates that you’re making a copy of the file rather than moving it.
4.  Right-click the tada.wav file on your Desktop and rename it test1.wav.
5.  Drag the file onto Sound Recorder to open it.
6.  On the left is a display giving the cursor position in seconds. Because we’re at the beginning of the file, it reads 0.00. On the right is the file’s total playback time, which in this case is 1.25 seconds. You can slide the cursor manually or use the Rewind and Fast Forward keys to shuttle to different parts of the file.
7.  Click the Play button and you’ll hear the file play back.
8.  Click the Effects menu, and you’ll see Sound Recorder’s list of editing effects you can apply. You can increase/decrease speed and volume, add echo, or reverse the playback. Because you’re working on a copy of a file (or a scratch file), go ahead and play with these editing effects to get a sense of how they can change the sound.

Editing a .wav File

The Edit menu provides the functions for editing your .wav file. You can copy the entire sound and paste it into any Windows application. You can also copy one sound, then open another .wav file, and paste that sound into the newly opened .wav file. Here you have two pasting options: Paste Mix or Paste Insert. As the names imply, the first mixes the two sounds together, and the second inserts the sound being pasted before the sound you have opened as a .wav file.

From the Edit menu, you can also mix in another sound (.wav file), or insert an additional .wav file, which will play immediately before the one you have opened. When inserting or mixing a second .wav file, use the sliding cursor to determine where to place the mixed or inserted file.
