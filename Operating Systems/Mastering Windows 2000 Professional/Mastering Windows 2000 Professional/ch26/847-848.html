Using the Telephony Settings

Sitting in the Control Panel is an icon labeled Phone and Modem Options. The telephony settings contained in this applet are global settings that you can use anytime you are using the modem to dial a service.

For example, imagine you will be calling from three primary locations: your home, which is a local call; your other office, which is a local call; and another location away from the office, which is usually a long distance call. You do not want to have to input the different local and long distance settings every time you change locations; instead, you can create different calling locations and store these parameters centrally.

You can also set the area code and country you are calling from when you are at a particular location. If you are setting your home location, you will probably leave these options blank. However, in many office environments, you need to dial a 9 to access a local line and 8 to access a long-distance line.

Finally, you specify the long-distance company access number and type in your calling-card number. This line is optional.

NOTE:  We recommend that you turn on the calling-card option on your laptop every time that you want to use it. If you preprogram these numbers and your laptop gets stolen or misplaced, your calling-card number is available for anyone to see.

If the location that you are dialing from has call waiting, you should turn off this feature because it may disrupt your modem communication. In a lot of locales, you dial *70 to turn off call waiting. (The code to disable call waiting may vary from locale to locale, so please check with your local telephone company for the correct code for your area.) You can preset this option on the next line of this dialog box.

The last item in this dialog box enables you to select whether you are dialing from a location that supports tone dialing or rotary dialing. If you are at a location that has only rotary dialing, select the pulse dial option.
