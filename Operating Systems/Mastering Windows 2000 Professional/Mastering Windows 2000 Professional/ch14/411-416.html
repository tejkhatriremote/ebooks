Cover Pages

The second tab of the Fax Properties dialog box is the Cover Pages tab. This dialog box is Windows 2000’s way of helping you organize your cover pages. If you’re like most people, you have several cover pages. Probably, you have one for formal business communications, one for personal business communications, and one just for fun. Maybe you have more than this, maybe less. Whatever your personal needs, the tab shown in Figure 14.4 will help you to keep them organized.


Figure 14.4  The Cover Pages tab of Fax Properties

The list box that contains the list of cover pages available for use occupies the main portion of the tab. When you first start to use the Fax Service, this list will be empty. We will go through the steps to create a cover page in the section entitled “Using the Cover Page Editor.”

The Cover Pages tab has four buttons that you’ll be using:

Open
This button opens the highlighted cover page in the list. When you highlight a cover page and click the Open button, the Cover Page Editor is opened with that page already loaded and ready for editing.
New
The New button opens the Cover Page Editor with a blank cover page loaded, ready for you to create your own page.
Add
Just as the name implies, this button enables you to add a new page from file. The difference between the New and Add buttons is that the Add button adds a previously created page to the list, whereas the New button creates a page from scratch.
Delete
The last button removes a currently installed cover page from the list.

Status Monitor

The next tab of the Fax Properties dialog box is the Status Monitor tab. This tab controls the behavior of the Fax Status Monitor, a tool that will help you monitor the activity of the Fax Service. The Status Monitor tab is shown in Figure 14.5.


Figure 14.5  The Status Monitor tab contains settings that control the Fax Status Monitor.

The Fax Monitor lists the status of faxes being sent or received, and records fax-related events in the log files. The tab contains the following settings:

Display the Status Monitor
Determines whether or not the Status Monitor is displayed when sending or receiving faxes in Windows 2000.
Status Monitor Always on Top
Tells Windows 2000 to always display the Fax Status Monitor on top of other windows when it is open.
Display Icon on Taskbar
Tells Windows 2000 to display the icon for the Status Monitor in the System Tray of the Taskbar. This can make it very easy to get to the Fax Queue whenever you want to view the status of the Fax Service.
Play a Sound
This setting causes Windows 2000 to play a sound when a fax is received. You can change the sound file in the Sounds and Multimedia applet in Control Panel.
Enable Manual Answer for the First Device
This setting can be very useful if you share the phone line between your phone and your fax modem. With the setting checked, you will be prompted to answer with the fax modem when an incoming fax is detected. Otherwise you will be able to answer the phone normally.

It is interesting to note that the Properties keep referring to the Status Monitor as a separate tool, yet when you send or receive a fax, the tool displayed is the Fax Queue. You can also start the Fax Queue by clicking its icon in the Fax group on the Start menu.

Advanced Options

The final tab to configure is the Advanced Options tab. This tab provides access to some of the management tools for the Fax Service. The Advanced tab is shown in Figure 14.6.


Figure 14.6  The Advanced tab of the Fax Properties

Let’s take a look at each of the options on this tab.

Fax Service Management Console

The first option on the Advanced tab is to Open the Fax Service Management Console. This opens the Microsoft Management Console (MMC) with the Fax Service snap-in loaded as shown in Figure 14.7. The Fax Service Management Console gives you a one-stop location to manage the settings for your installed fax devices and for the log settings for sending and receiving faxes.


Figure 14.7  The Fax Service Management Console

The console tree is comprised of entries for the Fax Devices and Logging settings. To view two fax devices installed in your Windows 2000 system, click the Devices entry in the left pane, and the installed modems are shown in the right pane. Likewise, to view the logging options, click Logging in the left pane and the log options are displayed in the right pane.

You can modify the fax properties of a particular fax device by double-clicking its icon in the right pane, or right-clicking the icon and selecting Properties. The Properties dialog box has two tabs, General and Received faxes.

On the General tab (see Figure 14.8), you can select whether the fax device will send, receive, or both. It’s possible that you would want to have one fax device to receive incoming faxes, and another to send your outgoing faxes for greater efficiency. Windows 2000 makes this easy by enabling you to make those selections here in the console.


Figure 14.8  The General tab of the Fax Device Properties lets you choose whether to send or receive.

There are two sections on the General tab. The first is the Send section, and it determines if the device will send faxes and the Transmission Station Identifier (TSID) of the fax device. The TSID is usually the fax number of the device, and it will be displayed on the cover sheet automatically. The TSID may also be displayed in the Fax Queue when you are receiving a fax.

The second section is the Receive section. The options here are essentially the same; a checkbox tells the Fax Service that this device will receive faxes. This section has the Called Station Identifier (CSID) setting also, which determines what will be displayed in Event Viewer where the received fax events are logged. The last setting in this section is for the number of rings to wait before answering the incoming fax call.

The Received Faxes tab lets you specify what the Fax Service will do with incoming faxes. The first option, To Print On, lets you direct all incoming faxes on this device to a printer. The next option, Save in Folder, defaults to the Fax folder in your user profile. After installing the Fax Service, you’ll notice that there is now a Fax folder inside your My Documents folder. You can specify any folder you want to use to store the incoming faxes by changing the path.
