Advanced Cover Page Editing

Well, now you can create a simple cover page that contains all of the necessary information. But just having the basic information is, yawn, so dull. You need to spice it up a bit with some graphics.

The Drawing toolbar has some simple graphics tools that enable you to add some visual interest to your cover page. The shapes and text buttons are very much what you would find in the Paint program included with Windows 2000, but the Editor has also included some tools like Send to Back and Send to Front that make it simple to create some great layered looks. Figure 14.16 shows some basic shapes and fill styles added to our sample cover page.


Figure 14.16  You can add simple graphics to your cover page to add interest.

Here are the buttons you’ll be using in the Cover Page Editor:

Select  This chooses the select tool to manipulate items in your page. This is the standard arrow cursor that you can use to select and move items on the page.
Text  This button changes the cursor to a cross-hair style and lets you draw a box in which to place text.
Line  This button lets you draw lines of different types based on the Format settings.
Rectangle  Draws a simple rectangular shape. If you configure the fill settings before drawing, the rectangle can be filled or colored.
Rounded Rectangle  Same as the Rectangle except that the corners are rounded off.
Polygon  The Polygon tool lets you draw, well, polygons. That may not be too clear, but each time you click your mouse button you will define a vertex of a polygon. When you are satisfied with the shape, double-click the last “corner” to stop drawing.
Ellipse  This button enables you to draw circles or ovals.
Bring to Front  This command moves the selected field or object to the front of the picture, or to the top of the stack of images.
Send to Back  Just the opposite of Bring to Front. This command sends the selected object to the back of the picture.
Space Across  Evenly spaces the selected objects across the page horizontally.
Space Down  Like Space Across, but Space Down evenly distributes the selected objects vertically on the page.
Align Left  Organizes the selected objects to the left of the page.
Align Right  Lines up all of the selected objects at the right side of the page.
Align Top  Aligns all of the selected objects at the top of the page.
Align Bottom  Organizes all of the selected objects from the bottom of the page.

These commands will make it possible for you to create very complex, or very simple, cover pages for your faxing needs. When you have finished your work of art, be sure to save it so that it will be available in the future for you.

You should take note of the message on the Cover Pages tab of the Fax Properties dialog box. If your network administrator has decided that only the official company cover pages are going to be allowed, they can be set to override your own personal cover pages. This can be done through the properties for each fax device in the Fax Service Management Console.

This can be a very handy setting to use if you are the one sharing out the fax device and want to be certain that people are using the right cover pages. If you want to make sure that anyone using your fax device uses your cover pages, you will need to make that setting in the Fax Service Management Console. Click Start Programs Accessories Communications Fax Fax Service Management.

Right-click the Fax Service on Local Computer entry in the left pane, and select Properties from the context menu. Figure 14.17 shows the Fax Service on Local Computer Properties dialog box. This dialog box has two tabs, General and Security.


Figure 14.17  The Fax Service on Local Computer Properties

The first section on the General tab is the Retry Characteristics section. Here you can determine the number of times the service will attempt to send a fax if it doesn’t succeed the first time. You can set the number of minutes between each retry and the number of days that the unsent fax will be kept in the queue.

The Send Settings section contains several settings that determine how a fax is sent. The settings are

Print Banner on Top of Each Sent Page
Prints a banner containing the sender’s information at the top of each page.
Use the Sending Device TSID
Tells Windows 2000 to include the specific TSID for the fax device that was used to send the fax.
Don’t Allow Personal Cover Pages
Tells Windows 2000 that users must use the cover pages defined on the local computer instead of submitting their own.
Archive Outgoing Faxes In
Defines the path where all of the outgoing faxes will be copied. This is particularly useful if your organization needs to maintain copies of all documents.
Discount Period Starts/Ends
This setting defines the period of time when the lowest prices exist for using the phone lines for the fax service.

The Security tab contains the permissions settings for the Fax Service. You can fine-tune the permissions for users or for groups defined either on the local computer or in your domain. The Security tab is shown in Figure 14.18.

Summary

You have now learned how to use the Fax Service in Windows 2000. You learned how to install and configure the service, and how to use the Cover Page Editor to create a custom cover page. We also discussed how to use the Fax Service Management Console to control the properties of the Fax Service.
