The Shortcut Key allows you to assign a key combination that will run the shortcut from anywhere. For example, to jump to My Computer without minimizing all your other windows first (you do need to minimize the other windows to complete this procedure, though):

1.  Get to the Desktop. Then right-click My Computer and choose Create Shortcut. This creates a new shortcut on the Desktop called Shortcut to My Computer.
2.  Right-click the new shortcut and choose Properties.
3.  In the Properties dialog box, click the Shortcut tab then click in the Shortcut Key field.
4.  Press Ctrl+Alt+C to assign the shortcut key to Ctrl+Alt+C.
5.  Click OK to finish.

Now whenever you want to open My Computer, just press Ctrl+Alt+C, which does take a little manual dexterity, but is quick. Use this same trick for any object you use regularly and often need to fish around to open.

The Run field in a shortcut’s properties sheet determines whether the object opens in a maximized, minimized, or normal (floating) window.

Changing Property Settings from My Computer

Probably the most powerful of the properties sheets can be reached directly from My Computer. Clicking the My Computer icon on the Desktop and choosing Properties brings up the System properties sheet, shown in Figure 4.3.


Figure 4.3  A grand overview of your computer’s attributes is available by right-clicking My Computer and choosing Properties. Use some caution with these settings. This box is also available from the System applet in the Control Panel.

Examine the five tabs in Figure 4.3. The General tab tells you some useful information about the version of Windows you are running, how much memory your computer has, and what type of CPU chip is in your machine. This will come in handy the next time someone asks you what’s in the computer you’re running. Instead of drumming your fingers on the desk and feeling like a doofus, you can open this box and read what it says.

NOTE:  The General tab of System Properties is one of the only places where you can see which Service Packs, if any, have been installed on the system.

The Network Identification tab displays the full name of your computer as it would appear in the table of your local Domain Name System (DNS) server (see Figure 4.4). It also displays the name of the domain to which your computer belongs. The Network ID button starts the Network Identification Wizard. The Properties button gives you direct access to the network identification settings if you are a more experienced user.


Figure 4.4  The Network Identification tab contains information on your computer name and Domain.

The Hardware tab provides access to a number of different hardware management tools in Windows 2000 (see Figure 4.5). The first of these tools is the Hardware Wizard, which enables you to add, remove, or troubleshoot hardware in your computer. Device Manager is new in Windows 2000, and although it has the same name as the tool in Windows 98 it looks a bit different in Windows 2000. In Windows 2000, the Device Manager is a snap-in for the Management Console, so when you start Device Manager, it opens in a separate window with the same basic interface as the other administrative tools. The last part of the Hardware tab is Hardware Profiles. Hardware Profiles are very helpful if you use your computer in different configurations and desire an easy way to shift between them.


Figure 4.5  The Hardware tab gives you access to the main hardware configuration tools in Windows 2000.

The User Profiles tab lets you save different configurations of your Desktop appearance, including which items appear on your Start menu and other interface settings, so that different Desktop appearances can be associated with different users logging into the system (see Figure 4.6). You can also specify a “roaming” configuration, which permits your Desktop appearance to be available to every machine on your network. This way you can work at other machines using your own familiar interface.


Figure 4.6  The User Profiles tab lets you manage the profiles stored on the local computer.

Finally, the Advanced tab of System Properties lets you change some of the performance and recovery options for Windows 2000 (see Figure 4.7). The Performance button enables you to set the size of the page file and the Windows 2000’s virtual memory file, and it lets you optimize performance for either desktop applications or background services. The Environment Variables button lets you set the various system or user level variables, such as the location and name of the temporary folder or the system path statement. The last button is the Startup and Recovery button. This button enables you to determine how the computer will start up and what to do if a critical error occurs.


Figure 4.7  The Advanced tab enables you to change performance and recovery options for Windows 2000.

WARNING:  As a rule, don’t mess with the settings in the My Computer’s System properties sheets unless you know what you’re doing. Adjusting the virtual memory settings, for example, will more likely negatively impact your system’s performance than accelerate it. Examining all the pages is fine if you click Cancel after viewing the information.

Summary

In this chapter, we learned how to adjust properties for many of the objects in Windows 2000 Professional. Whether you prefer to use the context menu by right-clicking an object, or highlighting an object and pressing the special Properties key on your Windows 95/NT keyboard, Properties settings are easily accessible in Windows 2000.

Properties sheets are available for many parts of Windows 2000 right from the Desktop. Right-clicking My Computer and selecting Properties gets you to the most important of these settings, the System properties sheets.
