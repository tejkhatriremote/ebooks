Scanning Images

If you have a scanner, you should be able to configure the Imaging accessory to let you scan documents. All you need is a Windows 2000-compatible (32-bit) TWAIN driver for your scanner. Unfortunately, it may not be easy to find an NT or Windows 2000 scanner driver. In fact, you may have trouble finding TWAIN drivers for DOS and 16-bit Windows, and they’ve been around much longer than NT has. Over time, this will cease to be a problem, but do keep in mind what you’ll need if you go shopping for a new scanner. Windows 2000 offers much better support of scanners than NT 4 did, so this should be less of a problem.

Scanners currently come in SCSI, video interface (IEEE 1394 or FireWire), Universal Serial Bus (USB), and parallel-port flavors. The video-interface ones, in particular, require you to install a special card in your computer. USB devices are relatively new to the market, but would be the first choice of most users because they are so easy to use. With USB, you simply plug in the device, Windows 2000 recognizes it, and the drivers are installed.

Assuming you have a scanner and have successfully installed both it and its TWAIN driver for Windows 2000, you’re almost ready to scan. Make sure your scanner is connected and turned on. (Note that some scanners require that you turn them on before you start your computer.) Then, choose File Select Scanner from the Imaging menu bar and select your scanner model from the list that appears. (If you don’t see your scanner listed, double-check that your scanner driver was installed successfully.)

You can configure the scanner to produce different types of scans by selecting File Scan Preferences. You can choose one of the three predefined options or select Custom and click the Settings button. In the Custom Scan Settings dialog box, shown below, select the desired number of colors and the compression type. The .jpg type also lets you select a low, medium, or high resolution, and the black-and-white .tif types give you the option of using reversed bit order. In most cases, you shouldn’t change the default settings, although most .tif readers (including this Imaging accessory) know how to read files using either bit order.

To scan a page, follow these steps:

1.  Make sure you’ve put the page in the scanner’s sheet feeder, or the flatbed (if your scanner has one).
2.  Choose File Scan New.
3.  Depending on which features your scanner supports, the next dialog box contains options for single-page or batch scanning, single-sided or double-sided scanning, and so on. Choose the desired settings and press Enter to begin the scan.

You can delete individual scanned pages by selecting them (press Ctrl+spacebar) and choosing File Delete. Or you can rescan a page by selecting Page Rescan. The Insert and Append options on the Page menu let you insert or append pages either by scanning or by using preexisting files.

Annotating Images

The Annotation menu provides several handy tools. To begin with, you can make these tools more quickly accessible by choosing Annotation Show Annotation Toolbox. The tools include a highlighter (for marking regions without blotting them out), tools for drawing freehand and straight lines, and tools for drawing hollow and filled rectangles. Hollow rectangles are great for marking off paragraphs or part of a column. Filled rectangles can be used for redacting (blotting out regions of a document, usually for security purposes).

The toolbar and menu also include three text tools:

Attach a Note
Enables you to attach text notes to a document; the notes can be redisplayed in a separate window by clicking the little note symbol that appears below the page.

Typed Text
Lets you type text, which will be merged graphically onto the image.
Text from File
Lets you add text from a file to the image.

Finally, there’s a Rubber Stamp tool, which lets you place one or more predefined “rubber stamps” on a page. You can define new stamps using either text or an image from a file. Figure 11.9 shows the creation of a new rubber stamp using a graphics file.


Figure 11.9  You can create realistic-looking seals and “rubber stamps” using image files.

TIP:  Because you can select a region of an image and save it as a separate file, it is easy to scan in a sheet containing actual rubber stamps (or seals), then save each imprint as an image, and create an Imaging rubber stamp from each one. When you apply these Imaging stamps, they look very much like the result of using a real rubber stamp.

Unfortunately, the Imaging accessory does not include an Undo option. However, if you do not choose Make Annotations Permanent, you can avoid saving any annotation changes you’ve made.

Summary

In this chapter, we introduced you to a few of Windows 2000’s more useful utilities. Notepad is a basic text-editing tool that is useful for modifying system configuration files, writing Web pages, and even typing a simple letter. Character Map is the tool to use when you want to insert that special symbol into your document. And the Imaging utility is great for viewing faxes you have received or scanning new pictures into your Windows 2000 computer.
