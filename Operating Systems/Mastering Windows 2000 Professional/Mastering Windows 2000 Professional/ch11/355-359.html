Using Imaging’s File Functions

As you can see in Figure 11.6, the Imaging accessory is equipped with the standard File menu, along with its other pull-down menus in the menu bar. The Open command on the File menu is the obvious way to open existing image files.


Figure 11.6  The Imaging window

Several types of image files are supported, including .bmp, .jpg, .pcx, .dcx, and .tif. You can see the available types in the Files of Type list, shown below.

Thanks in part to Web browsers, .jpg has become a common format for exchanging compressed color images. The .dcx format is a multipage version of .pcx, which you may remember is a format popularized a while back in applications such as Paintbrush, and it is still supported in the MS Paint accessory. The .tif format is an imaging umbrella-standard that was hashed out several years ago by Aldus, Microsoft, and a number of other companies; it supports several types of color and black-and-white Tagged Image File Formats.

NOTE:  The most significant format for document-imaging purposes is either Group 3 or Group 4 compressed bi-level TIFF. Group 3 uses Huffman encoding to compress black-and-white images by using short bit patterns to stand for much longer runs of black-and-white dots. Group 4 uses the same technique but adds the twist of encoding only the changes from one line to the next. A variant of Group 3 is the compression standard used in fax machines and fax modems. Group 4, because of its superior compression rates, is the format of choice for most top-of-the-line document imaging systems. Why such use of compression? Keep in mind that a single, letter-size document scanned at 300 dots per inch takes up more than one megabyte of memory. Using Group 4 compression, it is not uncommon to have these same files take only 30K to 60K, or roughly one-fifteenth to one-thirtieth of their original size. Compression becomes crucial both in the areas of storage (disk and RAM) and transmission (for example, over the Internet).

Besides supporting several of the top document image formats, the Imaging accessory also lets you add new pages to a file, delete pages, insert pages, and save files to disk in different formats and at different resolutions and numbers of colors. For example, you can open a 1200dpi (dots per inch) color image and save it as a 200dpi black-and-white image. This would be identical to a good-quality faxed document, and it would take vastly less time to e-mail, even using a slower modem connection.

To add or insert new pages, simply chose Page Insert, or Page Append. As you can see in Figure 11.7, both options let you insert or append the new page by scanning it or by taking it from a preexisting image file.


Figure 11.7  Adding or inserting a page is as simple as scanning it in or loading it from another file.

Printing an open image file is a matter of selecting File Print. And faxing can be accomplished in the same way, provided you have installed a third-party faxing solution.

E-mailing of image documents is identical to e-mailing in Microsoft Word, WordPerfect, and other Windows applications. Simply choose the File Send menu option.

Viewing Documents

Imaging’s display of documents works quite nicely. You can view thumbnails, a full-size page, or a combination of thumbnails and a full-size page, as shown in Figure 11.8. Thumbnails refer to the tiny, scaled-down, postage-stamp-sized images, which ideally let you see just enough of each page to recognize it. Single-clicking a thumbnail immediately displays it in the larger, full-size window.


Figure 11.8  Thumbnails help you access the pages of a multipage image file.

As you may be aware, displaying detailed bitmap images on most monitors requires that they be scaled down quite a bit to fit on the screen, and this results in a loss of detail in the screen display. You may need to play with the combination of the Imaging accessory’s window size and the scaling factor to achieve an acceptable balance between seeing a large enough region of a document and having the thumbnail show enough detail. You can adjust the size and default shape (letter size, legal size, or a size that matches the image shape, for example) of thumbnail images by selecting View Options Thumbnail.

Unfortunately, one thing you cannot do (at least currently) is have multiple separate files open simultaneously in the same viewer. So thumbnails are useful only when you open a file containing multiple pages. Note that you can, however, have multiple Imaging instances opened simultaneously, and although dragging and dropping between them doesn’t work, copying and pasting pages from one to the other works fine (see Chapter 8, “Sharing Data between Applications,” for details on copying and pasting).

Imaging also has a handy right-mouse-button menu, which includes some display options. Right-clicking the main viewing area displays a pop-up menu with options for zooming, fitting, rotating, flipping, printing, and rescanning.

Right-clicking thumbnails gives you a similar but smaller menu.

If the image is larger than the display area, you can also right-click and push or drag the image around (a hand cursor appears when you’re able to drag the image).

TIP:  You will have much better results with viewing image documents if you set your display resolution to the highest setting it supports. The higher the screen resolution, the more dots there are to display your images, and hence, your documents will show more detail and be much more readable. See Chapter 5, “Customizing Your Desktop via the Control Panel,” for more information.
