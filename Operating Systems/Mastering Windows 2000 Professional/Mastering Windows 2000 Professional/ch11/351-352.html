Using Character Map

Here’s how to use Character Map:

1.  Run Character Map (it’s in the System Tools group). The Character Map dialog box comes up, showing all the characters included in the font selected in the Font list (a font can contain up to 224 characters). Figure 11.5 shows an example.


Figure 11.5  The Character Map table shows all characters that are in the font selected in the Font list.

2.  In the Font list, choose the font you want to work with. Most of the fonts have the same characters, but some special fonts have completely different character sets. For example, the Symbol font includes all sorts of special math and Greek symbols; the Wingdings font consists of a wacky set of little pictures.
3.  To make it easier to see the individual characters, you can click a character box and hold the mouse button down to magnify the symbol. You can accomplish the same task with the keyboard by moving to the character using the arrow keys. With this technique, each character is magnified as you select it.
4.  Double-click a character to select it, transferring it to the Characters to Copy box. Alternatively, after you’ve highlighted a character, you can click the Select button or press Alt+S to place it in the Characters to Copy box.

NOTE:  You can change fonts at any time. Just be aware that the fonts of the characters you previously placed in the Characters to Copy box will also change—not just new characters.
5.  If you want to grab more than one character, keep adding them in the same way. Each new character is added to the end of the string in the Characters to Copy box.
6.  Click the Copy button. This places everything that is in the Characters to Copy box onto the Clipboard.
7.  Switch back to your destination application and use the Paste command (typically on the application’s Edit menu) to insert the characters into your document. You may then need to select the inserted characters and choose the correct font to format the characters correctly.

After you’ve entered a character in this way, you’re free to change its font and size as you would any character you typed.

Entering Alternate Characters from the Keyboard

Notice the word Keystroke at the bottom of the Character Map dialog box. When you click a character in Character Map, this area displays the keys you would press to enter the character directly from the keyboard rather than from Character Map. For the characters in the first three rows—except the very last character in the third row—the keystroke shown will be a single key on your keyboard.

TIP:  You won’t have any success entering special Alt-key+number combinations with the Character Map until you remember that the Character Map is looking for input from the numeric keypad part of your keyboard and not from the row of numbers above the letters you type. The numeric keypad is the cluster of keys on the right side of a standard desktop keyboard.

If you’re working with a nonstandard font, such as Symbol, pressing the key shown enters the selected symbol into your document.

For all the other characters, Character Map instructs you to enter a sequence of keys in combination with the Alt key. For example, say you want to enter the copyright symbol © into a document. Note that with a standard text font such as Arial or Times New Roman selected in Character Map, the program lists the keystrokes for the copyright symbol as Alt+0169. Here’s how to enter the character from the keyboard:

1.  Press Num Lock to activate the numeric keypad on your keyboard if the keypad is not already active.
2.  Press Alt and as you hold it down type 0169. When you release the Alt key, the copyright symbol should appear in the document.

Not all Windows programs accept characters in this way, but it’s worth a try as a shortcut to using the Character Map.
