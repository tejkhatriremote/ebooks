Setting Up a New Disk Drive in Windows 2000

If you’ve added a new disk drive to your system, Disk Management will be able to see it, but you’ll need to create a partition and a volume on it before Windows 2000 can use it.

NOTE:  After you create a partition on a new disk and format it, you can use it in Windows 2000 without rebooting. The new dynamic volumes in Windows 2000 will even let you change the size of a partition or extend it across several hard drives without rebooting.

Let’s step through adding a new partition and creating a volume on a new disk:

1.  In the Disk Management window, click the new disk. From the Action menu, select Create Partition. This will open the Create Partition Wizard.
2.  Select the type of partition you want to create, either primary or extended. By default, Windows 2000 assigns all available space on a disk to its primary partition. To create other extended partitions on the disk, set the size of the primary partition smaller. For example, you’ve added a 1GB hard drive and you want to create two 500MB partitions. Select 500MB as the size for your primary partition. This leaves the other 500MB of space free for an extended partition that you’d create later.

NOTE:  Windows 2000 supports a maximum of four “basic” partitions per hard drive. Windows 2000 supports only one extended partition per drive, but it allows as many as four primary partitions. Basic partitions are the same as traditional partition types found under earlier version of NT or MS-DOS. The new dynamic volumes allow as many logical volumes per disk as you want.
3.  After setting the partition size, click OK. The primary partition has now been created.
4.  To format the new partition, click the drive and select Format from the Action menu.
5.  Select the type of file system you want to use—FAT, FAT32, or NTFS. Remember, FAT will give you somewhat better speed performance but with limited volume size; NTFS enables you to use Windows 2000 file compression, security features, and file recovery.
6.  After selecting the file system, you can enable/disable the QuickFormat feature. Disabling QuickFormat tells Windows 2000 to first scan the partition for bad sectors. If you know the disk is free of bad sectors, go ahead and QuickFormat it. If you’re not sure, it’s better to have Windows 2000 scan for bad sectors. It will take considerably longer than a QuickFormat, but could save you trouble up the road.
7.  Click OK. When Windows 2000 asks you to confirm your choice, click OK again. Windows 2000 begins the formatting process. The time depends on the type of format and the partition size.
8.  When formatting is complete, Windows 2000 assigns a drive letter to the new volume, and you can begin using it.

WARNING:  You can apply the Set Active command to a newly created partition. Use this command only if you want your machine to boot from this new partition.

Creating Extended Partitions and Logical Drives

The main difference between a primary partition and an extended partition is that you can create multiple logical drives only on an extended partition. So what’s a logical drive? Simply put, a logical drive is a designated portion of a physical drive. Windows 2000 is aware of two kinds of drives, physical and logical. A physical drive is the actual drive itself, and Windows 2000 can create partitions on it and format it. You can create a logical drive on an extended partition that may be only part of a physical disk, but Windows 2000 will see that logical drive as a separate unit, and you can map a drive letter to it.

Let’s say you’ve added a 2GB hard disk to a dual-boot system, and you have created a 1GB primary partition and formatted it using FAT (so that Windows 3.1 can see it). You’ve decided to make an extended partition, using the remainder of the hard drive (1GB). You’ve also decided to create two logical drives that will use the NTFS file system. The steps that follow take you through creating an extended partition and logical drives.

NOTE:  Only extended partitions can contain logical disk volumes.
1.  Right-click the portion of the new hard drive labeled Free Space or select the region labeled Free Space, and from the Action menu, choose Create Partition. In the Create Partition Wizard, select Extended as the partition type. If DOS clients aren’t going to be able to access the newly created partition, Windows 2000 will warn you of this fact and verify that you want to continue. Click OK. Windows 2000 now creates your new extended partition.
2.  Next, right-click the newly created extended partition—or select the region labeled Free Space—and from the Partition menu, choose Create Partition again.
3.  Windows 2000 now prompts you for the size of the logical drive, defaulting to using all available space—1024MB. For our example we’ll create two 512MB logical drives, one at a time; so specify the size of the logical drive as 512MB and click OK. Windows 2000 now creates the logical drive and assigns it the next sequentially available drive letter.
4.  Right-click the new logical drive—or click the Partition menu—and select Commit Changes Now. This step is necessary before you can format the drive. Windows 2000 saves your disk information to the hard drive on which the operating system is installed and then advises you to update the Emergency Repair Disk you created during initial installation. More on that later.
5.  To format the newly created logical drive, right-click it—or click the Action menu—and select Format. Windows 2000 prompts for which file system you’d like to use. Select NTFS and leave the QuickFormat box unchecked.
6.  Windows 2000 now scans for bad sectors and then begins formatting. When the format is complete, Windows 2000 displays a window with disk and volume space information.
7.  To create the second logical drive, repeat steps 2 through 6.

TIP:  If you want to create an Emergency Repair Disk (ERD) outside of Disk Management, open Administrative Tools and run Backup. Windows 2000 Backup includes a wizard that will enable you to create a new ERD.
