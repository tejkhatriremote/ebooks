To see the access rights for a particular group, in Local Security Policy, expand Local Policies User Rights Assignment. The list of User Rights, as they are assigned by default in Windows 2000 Professional, are shown in the right pane (see Figure 30.18).

Windows 2000 also has Advanced Rights, most of which are intended to be enabled or disabled by your network administrator. Table 30.7 lists these rights and the privileges they grant.

NOTE:  Although Windows 2000 Professional has many server-like functions, remember that each user logged on to your machine introduces a slight performance hit, especially if users frequently access files on your hard drive. You may want to set an alert in Performance so that when a certain number of connections is exceeded, Performance either notifies you or writes an event to the event log.

Creating a New User

Creating a new user account is easy in Windows 2000 Professional. We went through the process using the Add New User Wizard when we created the trapdoor account earlier in this chapter. Now, let’s step through the process using the Advanced options in Local Users and Groups. To do so, follow these steps:

1.  Open Users and Passwords in Control Panel and on the Advanced tab, click the Advanced button. This opens the Local Users and Groups console shown in Figure 30.19.
2.  Right-click the Users folder in the left pane, and select New User from the context menu.
3.  In the New User dialog box, type in the username that the person will use when logging into your computer.
4.  In the Full Name field, type the user’s first and last names. Enter a description of the account in the Description field.
5.  Type in the initial password, and then type it again in the Confirm Password field.
6.  It is a good idea to keep the default option selected for User Must Change Password at Next Logon since this means only the user will know their password. If you want to maintain more control over the users on your computer, uncheck this box. If this option is deselected, the options for User Cannot Change Password, and Password Never Expires are made available.
7.  When you are satisfied with the options you have selected, click the Create button to create the user account. If you don’t want to create any more accounts at this time, click the Close button to close the New User dialog.

When you’ve set all the new user’s properties, that account is ready to use.
