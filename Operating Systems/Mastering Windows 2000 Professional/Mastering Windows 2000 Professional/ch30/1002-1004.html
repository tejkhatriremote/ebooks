Task Manager Performance

The Performance tab (see Figure 30.30) enables you to monitor your computer’s usage of memory, processor time, and other resources. The two central graphs on the page measure the amount of memory used and the amount of processor time used. For multiple processors, you can choose to show one graph for each CPU or one graph for combined CPUs. The Performance tab also contains information on how much physical memory is being used and how much memory the Windows 2000 Kernel is using.

TIP:  When you run Task Manager, a small green bar appears on your Taskbar, next to your clock. This is a CPU usage meter. Notice that the meter rises most when you are doing something very active, such as typing or switching between applications.

Summary

Windows 2000’s suite of administrative tools cover the gamut of system functions, ranging from providing networking settings to detailed information about the operating system’s main “plumbing.”

Most of the tools are intended to help troubleshoot system problems; using them in concert, you can gather sufficient information to remedy most system hiccups and even avert larger ones by catching them early.

Other tools are more for system maintenance and for adjusting settings such as network and remote access rights. Here’s where you get to play Head Honcho (uh, provided it’s, ya know, OK with the network administrator) in determining access rights for other users connecting to your system.

Play with Performance Monitor and watch the parts of your system. It’s a great way to learn about what’s really going on behind the curtain. Although no set of tools is a panacea for all problems, those in Windows 2000 will do a good job of helping you keep your system up and running.
