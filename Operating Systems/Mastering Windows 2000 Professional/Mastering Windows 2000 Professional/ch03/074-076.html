Edit

The Edit menu contains the familiar commands you would expect to find in the Edit menu of an average Windows application. These include the following:

Undo
Reverses or undoes the last action you completed. Even if you send a file to the Recycle Bin, Undo restores it to the folder you deleted it from. In Windows 2000, this command will usually indicate what the last action was. For instance, when you have deleted a file, this option will display Undo Delete.
Cut
Removes files and folders from their present location. You can paste them into a new folder or use Cut as a means of deleting them.
Copy
Copies selected files and folders so you can paste them elsewhere.
Paste
Pastes or inserts files and folders you have cut and copied from other folders into the selected folder.

NOTE:  Cut, Copy, and Paste work just like they do in a word processor or spreadsheet. These commands use the Clipboard ability in Windows 2000 to move files and folders on the system just like you’d work with words and phrases in a document.
Paste Shortcut
Takes a file you’ve copied and pastes a shortcut to the file (or folder) in the active folder.
Copy to Folder
Copies the selected files and folders to a new folder.
Move to Folder
Copies the selected file or folder to a new folder location, and then deletes the original.
Select All
Selects (highlights) the entire contents of a folder.
Invert Selection
Reverses the item selection in the current folder. If you have two items selected, Invert Selection will select all but the two files you originally selected.

View

The View menu enables you to change the way things are presented in Windows Explorer. The View menu includes all the functionality of the Views button on the toolbar, and much more. The commands are as follows:

Toolbars
This option enables you to decide which of the toolbars will be displayed in Windows Explorer.
Status Bar
This command determines whether or not the Status Bar is shown at the bottom of the Windows Explorer window.
Explorer Bar
This is one of the new options in Windows 2000. The Explorer bar is essentially the left pane of Windows Explorer. Here you have options like Folders, Search, Favorites, and History.
Large Icons
Shows the contents of the right pane in Large Icon view.
Small Icons
Displays the contents of the right pane in the Small Icons view.
List
Uses the List view for the right pane.
Details
Shows the contents of the selected folder along with separate columns for extended information such as type of file, size, date modified, and so on. This option can be highly customized in Windows 2000.
Thumbnails
Displays the contents of the selected folder as a miniature view of each file. The Thumbnails view is very useful for sorting through multiple graphics files.
Arrange Icons
Rearranges your icons after you’ve been moving them around. This command offers several different methods of arranging the icons.
Line Up Icons
This command leaves the icons in their current order, but aligns them with an unseen grid so they appear neater.
Choose Columns
This is another new option in Windows 2000. This command enables you to choose and arrange the information to display in Details view.
Customize This Folder
This command opens the Customize This Folder Wizard that enables you to select how the folder will be seen by others.
Go To
This command is essentially the same as the Go To command in Internet Explorer. The Go To command enables you to easily go to any of your recent locations on your computer, your local network, or the Internet.
Refresh
This command causes Windows Explorer to renew its display.

The View menu is the location for many of the new functions in Windows Explorer. You might be seeing some of this as a carry-over of Microsoft’s attempt to integrate Internet Explorer into the operating system as they did in Windows 98. It is, but it’s also subtler than that. There is more of a separation between your Desktop and the Internet in Windows 2000 than there was in Windows 98, but at the same time the Internet functionality is still available to users who want it.
