From left to right, here’s what each of the toolbar buttons do for you:

Back
Returns you to the previous folder location. This works just like the Back button in your favorite Web browser.
Forward
This button also works just like your Web browser in that it moves you forward one step in your browsing (assuming that you have already been there and moved back with the Back button).
Up
This button moves you up one folder in the hierarchy.
Search
This button opens the Search pane on the left side of Windows Explorer. You can use this Search pane just like the Search utility on the Start menu—to find files, folders, people, and computers.
Folders
This button is on by default when you open Windows Explorer. Its purpose is to display the folders view in the left pane.
History
The history button keeps track of where you’ve been lately both locally (on your own computer) and remotely (on the local network or the Internet). It displays the History information in the left pane of Windows Explorer.
Move To
Allows you to select the destination when moving files and folders using Windows Explorer.
Copy To
Enables you to copy the selected item to another destination on your local computer or elsewhere on your network or even the Internet.
Delete
This button will erase the selected object from the system. This option is not supported for objects like hard drives that cannot be deleted this way. The default method of deletion is to move the selected file or folder to the Recycle Bin, thus giving you a second chance to decide that you actually do want the file or folder.
Undo
This button will reverse the last action. If you just copied or moved a file, the Undo button will…undo that action.
Views
This button enables you to select the view you want to use in Windows Explorer. It presents the choices that were discussed in the previous section of this chapter.

The toolbar in Windows Explorer looks much cleaner and more streamlined than it did in NT 4. It also gives you more opportunities to customize the options you want, including adding buttons for things like Map Drive, Disconnect, Cut, Copy, and Paste. It also includes some typical Web functionality like Home, Stop, and Favorites. Figure 3.8 shows the Customize Toolbar dialog.

Using Explorer’s Main Menus and Right-Click Menus

With Windows Explorer, you get a lot more menu for your money. The menus in Windows Explorer are context sensitive, meaning that the options presented change based on what you are doing at the time. When you select a file, the options on the File menu will reflect the activities that are appropriate for working with files, and so on.

This section briefly describes all the main menu options and then provides an overview of the right-click menu options in Explorer.

File

The File menu contains commands for opening files and creating and administering folders. You can also access Properties from the File menu.

Open
Opens your selected file using its associated application. If the file you have selected does not have an associated application, you will be prompted to select the appropriate application to use when opening the file.

NOTE:  Not all files can be opened. Many of the files you see in Explorer are system files that are used by the operating system or by applications. As a rule, files that can be opened have the extension .exe, or are represented by easily recognizable icons.
Print
Enables you to send a file directly to your default printer (only presented when a file is selected).
Open With
This option appears if you do not have a program associated with a particular file type (only appears when you have a file selected).
Explore
This option enables you to use an Explorer view of the selected folder, which seems a bit redundant given that you’re already in Windows Explorer (appears when you have selected a folder).
Search
Opens the Search utility in the left pane of Windows Explorer and sets the default location to search as the folder you have selected (appears when you have selected a folder).
New
Creates a new folder or shortcut in your current directory and gives them default names New Folder and New Shortcut. Also includes entries for most of your installed applications to create a new document of that type.
Send To
Sends your selected file or folder to any of the locations you have placed in your Send To folder. The Default choices are Floppy disk, Desktop (create shortcut), Mail Recipient, and My Documents.
Create Shortcut
Creates a shortcut to the selected file in the current folder.
Delete
Sends selected files and folders to the Recycle Bin.
Rename
Highlights the name of the selected file or folder and enters edit mode. Type your new name for the file or folder and press Enter to accept it.

WARNING:  Do not rename program or system files. For example, leave excel.exe and system32.dll alone. As a rule, only rename files and folders that you have created.
Properties
Displays the properties sheet for the file or folder. For more information on Properties, please see Chapter 4.
Close
Closes the active Windows Explorer window.
