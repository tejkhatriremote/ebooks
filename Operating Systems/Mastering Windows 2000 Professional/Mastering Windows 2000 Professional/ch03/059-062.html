Arranging Open Windows with the Taskbar

Like most objects you can click in the Windows 2000 environment, the Start menu and Taskbar offer right-click menus that enable you to set options for controlling their behavior and appearance. (The general topic of right-click menus is presented in depth in Chapter 4, “Setting Object Properties.” In this chapter you get to see some of the most immediate advantages of using them.)

You can quickly change the size and position of all open applications on the Desktop by making choices from the Taskbar’s context menu. Simply right-click anywhere on the Taskbar to bring up a menu with the following choices:

Toolbars
This new menu item enables you to add or customize the toolbars that appear on the Taskbar. The default toolbar presented is the Quick Launch toolbar with icons for Show Desktop, Internet Explorer, and Outlook Express.
Cascade Windows
Places each of your open applications across your Desktop, one almost on top of another. Your windows look somewhat like a deck of cards spread across a dealer’s table.
Tile Windows Horizontally and Tile Windows Vertically
Arrange your current applications in equal-sized windows so that you can see some contents of every window at once.
Minimize All Windows
Minimizes all your open applications to the Taskbar. This is probably the fastest way to clear the Desktop when it is cluttered, although the Show Desktop icon on the Quick Launch toolbar performs the same function.

TIP:  When you click a program that has been minimized (shows up as a button on the Taskbar), it returns to its former size and position on the Desktop.
Undo Tile and Undo Minimize All
Undo whatever Taskbar action you last performed. The first time you access the Taskbar properties, these commands do not appear.
Task Manager
This option is one of the ways to open the Task manager. Task Manager is useful for terminating an unresponsive application or getting a quick view of the system’s performance.
Properties
Brings up the Taskbar properties sheet, a tabbed dialog box with various pages of options. These options not only let you arrange items on the Start menu and the Taskbar, but also permit you to specify the opening and closing behaviors of those items.

The Properties for the Taskbar have some added features in Windows 2000. Now you can control which options are presented in the Start menu by checking the options in the Properties. This means that you can choose to display the Administrative Tools group, to expand the Control Panel, and to enable the new personalized menus.

Customizing the Start Menu and Taskbar with the Taskbar and Start Menu Properties Sheet

Right-clicking the Taskbar and choosing Properties from the menu that pops up displays the Taskbar and Start Menu Properties sheet, shown in the following illustration. This sheet (a dialog box, really) contains: General and Advanced. A better title for the latter tab might have been Programs on the Start Menu. This would reduce the confusion you might have when you first see it. (“Hmm, so is this where I go to start all my menu programs? I thought that was what the Start menu was designed for!”) But now that we’ve made a point about it, you’ll never be confused again. Let’s take the tabs one at a time.

The General Tab

The General tab of the Taskbar and Start Menu Properties sheet lets you set a few basic options for the Taskbar and the Start menu, as follows:

Always on Top
Keeps the Taskbar visible at all times. This is the default setting. (Screen saver programs ignore this setting, however, as they blank the entire screen.)
Auto Hide
Makes the Taskbar disappear until such time as you move your mouse pointer to where the Taskbar is hidden. AutoHiding the Taskbar is useful if you have limited screen space, or if you just don’t like the look of the Taskbar.
Show Small Icons in Start Menu
Shrinks the size of the icons on the Start menu and removes the Windows 2000 Professional label that appears on the left side of the Start menu. If you have limited screen space or just want to reduce the size of the Start menu, use this option.
Show Clock
Adds or removes the clock that by default is displayed at the far end of the Taskbar.
Personalized Menus
Turns Windows 2000’s ability to remember your favorite menu options on and off. With Personalized Menus turned on, over time the only options displayed on your menus are the options you actually use. All of the other options will be hidden.

TIP:  Remember, you don’t need to set any options to reposition the Taskbar. At any time, you can simply click the Taskbar and drag it to whichever edge of the screen you like.
