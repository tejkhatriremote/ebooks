Setting Up a HyperTerminal Connection

Your most basic rule in communicating holds true in HyperTerminal: You have to know how to reach the other party. Before doing anything else, find the telephone number your modem must dial to connect to the information service, BBS, or computer you’re trying to reach. Be sure you have the number for modem communications—in printed material it may be labeled the modem or data number. Voice or fax numbers won’t work.

While you’re looking for the number, see if you can locate any details on the communications settings in force at the computer you want to connect to. Your system must be set up to match, as detailed later in this chapter.

Starting HyperTerminal

To access HyperTerminal, begin from the Start menu and choose Programs Accessories Communications HyperTerminal.

Click the HyperTerminal icon to run the program and prepare to set up a new connection. After a specific connection has been set up, you’ll be able to select its icon from the HyperTerminal folder on the Start menu to start a communications session with that connection.

Defining a New Connection

When you start from the HyperTerminal icon (rather than from an existing connection icon), the program asks you to define a new connection in the dialog box shown on the next page.

You can also set up a new connection once you’re working with the main HyperTerminal window by choosing File New Connection or clicking the New button on the toolbar to display the same dialog box.

Your first step is to give the new connection a name. Keep it simple—something like MCI or Bill’s Office PC will do. Then pick out an appropriate icon for the connection from the scrolling list and click OK to continue.

Entering a Phone Number

Next you must supply the phone number your modem should dial to make the connection. HyperTerminal displays the Connect To dialog box, shown here.

Type the phone number for the new connection. You have your choice of styles:

•  1-(510)-555-1111
•  1-510-555-1111
•  15105551111
•  555-1111 (if you are calling a local number)

In case you’re wondering, HyperTerminal ignores dashes and parentheses in the main phone number.

If the phone number for the new connection is local, typing in the phone number is all you have to do—Windows 2000 has already entered your settings for the country code and area code. If you’re dialing out of the area or out of the country, use the first two boxes to set the correct country and area code. And if you have more than one modem connected to your computer, choose the one you want to use for this connection from the list at the bottom of the dialog box. If you have set up your area code dialing rules correctly, HyperTerminal will be able to dial the number correctly. Refer to Chapter 12, “Introduction to Communications and Using Phone Dialer,” for more information on setting area code rules.

Click OK at the bottom of the Connect To dialog box when all the settings are correct.

NOTE:  If you haven’t set up your area code rules, Windows 2000 may try to dial the local 10-digit number as long distance. To avoid this, where it says Area Code, enter your area code. Then in the space provided for the phone number, enter the 10-digit version of the number you are dialing. The Telephony API (TAPI) makes its decision about long-distance dialing based on the area code you enter. If it sees your area code, it will dial the phone number without adding the country code and area code first.

Setting Dialing Properties

After you’ve set the phone number, HyperTerminal assumes you want to dial it and presents you with the Connect dialog box in Figure 15.3.


Figure 15.3  The Connect dialog box

At this point, it’s safe to click OK to see if the setup works. But don’t be too surprised if nothing happens. In many cases, you’ll need to change other settings before you can connect successfully.

Look at the area labeled Your Location. Here, you should select the choice that describes where you are at this moment. The first time you use HyperTerminal, the only choice available is the nondescript selection Default Location. This corresponds to the area code and country you entered when you set up Windows 2000 Professional, so if you haven’t moved, it may well be correct for those items.

If you use a portable computer, however, you may be thousands of miles from your default location. And besides, there are other considerations: Do you have to dial 9 to get an outside line? Are you dialing from a customer’s office so it will be necessary to use your telephone credit card? Or are you visiting a friend who doesn’t believe in high technology and never installed a push-button phone? If so, tone dialing won’t work.

To change any of these properties for the default location or to set up new locations from scratch, click Dialing Properties. You’ll see the dialog box shown in Figure 15.4.


Figure 15.4  The Dialing Properties command button opens the Phone & Modem Options dialog box.

NOTE:  The Phone & Modem Options dialog box controls dialing settings for the Phone Dialer accessory, too. Phone Dialer is covered in Chapter 12, “Introduction to Communications and Using Phone Dialer.”
