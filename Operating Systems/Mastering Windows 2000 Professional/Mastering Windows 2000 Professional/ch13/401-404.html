Creating and Sending Messages

Everything we’ve talked about so far in this chapter is about organizing Outlook Express. In this section, we explain how to use Outlook Express to do something actually useful. It’s easy to forget that Outlook Express isn’t really about profiles or address books—it’s a tool for sending and receiving messages. All its other features and functions are supposed to support those activities.

Here’s the procedure for sending a message through Outlook Express:

1.  Start Outlook Express from the menu bar or the Launch Bar icon.
2.  When the Outlook Express window appears, choose the New Mail command from the toolbar, or from the menu command File New Mail Message.
3.  When the New Message screen in Figure 13.13 appears, either type the names of the recipients in the To field or click the To button to select names from an address book. If you want to send this message to more than one person, use semicolons to separate the names.


Figure 13.13  Use this screen to compose e-mail messages to send to any Internet recipient.


•  If you want to send duplicate copies of this message, tab or mouse to the Cc field in the New Message box and type the names. Alternatively, in the Address Book dialog box, click the Cc button to select from the list.
•  If you want to send a blind copy of this message, use the All Headers command in the View menu.

4.  Click the Subject field and type a subject line for this message.
5.  Move your cursor to the large box at the bottom of the screen and type the text of your message. You can find more detailed information about formatting messages, adding inserts, and attaching files in the sections that follow.
6.  When the text and format of your message are complete and ready to go, use the Send command in the File menu or the Send button in the toolbar to place the message in the Outlook Express Outbox.
7.  After you have finished composing messages, open the Outbox folder and choose the Send and Receive command in the Tools menu or the Send/Recv button on the toolbar. If all the messages in your Outbox are for the same message service, you can choose that service from the Deliver Now submenu, but it’s generally easier to use the All Services command.
8.  After Outlook Express has uploaded your message, it automatically transfers the message from the Outbox folder to the Sent Items folder. You can move the message to a different folder by dragging and dropping it in the appropriate folder in the left pane.

Formatting Messages

The message editor in Outlook Express uses the same formatting tools as WordPad, including a toolbar with buttons to control font, character size, style, color, indentation, and justification. To change the format of all or part of a message, highlight the text you want to format and click the appropriate button in the toolbar.

Just because it’s possible to do all kinds of fancy formatting tricks with your messages, it doesn’t mean you have to use them in every message. In most cases, you send a message to communicate information, and a message with three colors and four typestyles doesn’t contribute anything to the reader’s understanding of your message—it’s just hard to read. Sending an e-mail message that looks like a ransom note does not convey a businesslike image, but it might be exactly what you want when you send a birthday greeting to a 10-year-old.

It’s also a good idea to think about the type of hardware and software the person receiving your messages will use. Unless the recipient is also using Outlook Express or another e-mail client that can read HTML messages, all your special effects will probably be lost because other mail readers may not recognize formatting information.

If your intended recipient is flagged as one that should only be sent messages in clear text, Outlook Express will send the message in clear text and ignore all of your creative work (other than the sheer poetry of your message, of course).

Attaching a File to a Message

In addition to the text of your message, you can also attach text or binary files to your e-mail. Outlook Express will include a list of your attachments in the Attach header window, not in the actual text of your message. It’s possible to drag and drop a file into your message, but even though it appears with the plus sign indicating that it is being copied to the message, the file will only appear in the Attach field.

You can, however, add pictures (.bmp, .jpg, or .gif formats), text from files, hyperlinks, and horizontal lines just as you would to any Web page. Since Outlook Express is capable of sending e-mail as HTML files, it supports the standard tags and techniques for creating Web pages.

Use these steps to include an attachment with your e-mail message:

1.  In the window of a message you’re composing, select Insert File Attachment to open the Insert Attachment dialog box shown in Figure 13.14.


Figure 13.14  The Insert Attachment dialog box lets you browse for a file to attach to your message.

2.  Browse for the file you want to attach to your message. When you locate it, click it to highlight it.
3.  Decide whether you want to attach a copy of the file or just a link to the file. If you want to send a link to the file, check the box Make Shortcut to This Item. Click Attach to include the file or the link.

The attachment should now show up in the Attach header field of the message window.
