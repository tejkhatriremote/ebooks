Removing Existing Programs

With time, more programs will be made removable via the Control Panel. This is because the PC software industry at large has heard much “kvetching” from users and critics about tenacious programs that once installed are hard to remove. Some ambitious programs spread themselves all over your hard disk like olive oil in a hoagie, and there’s no easy way of reversing the process and returning your system to a pristine state. The result is often overall system slowdown, unexplained crashes, or other untoward effects.

To this end, aftermarket utilities such as UnInstaller™ from McAfee have become quite popular. Uninstall utility programs monitor and record exactly what files a new software package adds to your hard disk and which internal Windows 2000 settings it modifies. It can then undo the damage later, freeing up disk space and tidying your Windows 2000 system.

In typical fashion, Microsoft has incorporated such a scheme into Windows 2000. Time will tell if its mousetrap is as good as the competition’s. Probably not, but it will be “close enough for jazz,” as the saying goes. If programmers write installation routines that work with Windows 2000’s Add/Remove Programs applet, we’ll all be in luck.

One of the nicest improvements in the Add/Remove Programs applet is the new interface for listing the applications that are installed. This new interface also tells you how large the application is on disk, when you last used the program, and even how frequently you use the program. This is really helpful when you’re running out of drive space and trying to decide which program to delete to free up some space.

Use of the uninstall feature of the applet is simple:

1.  On the first tab, select the program that you want to remove.
2.  Click the Change/Remove button.
3.  Answer any warnings about removing an application as appropriate.

This procedure also enables you to add or remove components from some applications, which explains the name of the applet. If an application supports adding and removing components, you are asked if you want to remove all components, add or remove components, or reinstall the entire application.

NOTE:  After the program is removed, you must reinstall the program from its source disks to make it work again. You can’t just copy files out of the Recycle Bin folder to their old directories because settings from the Start button—and possibly the Registry—will have been deleted.

Setting the Date and Time

The Date/Time icon lets you adjust the system’s date and time. The system date and time are used for a number of purposes, including date- and time-stamping the files you create and modify, and scheduling fax transmissions. All programs use these settings, regardless of whether they are Windows or non-Windows programs. (This applet doesn’t change the format of the date and time—just the actual date and time. To change the format, you use the Regional applet.) To set the date and time

1.  Double-click the Date/Time applet. The dialog box in Figure 5.11 appears.


Figure 5.11  Adjust the date, time, and local time zone from this dialog box. A shortcut to this box is to double-click the time in the Taskbar.

2.  Adjust the date and time by typing the corrections or clicking the arrows. Note that you have to click directly on the hours, minutes, seconds, or A.M./P.M. area before the little arrows to the right of them will modify the correct value.
3.  Next, you can change the time zone you are in. Who cares about the time zone, you ask? Good question. For many users it doesn’t matter. But because people fax to other time zones, and some programs help you manage your transcontinental and transoceanic phone calling, it’s built into Windows 2000. These programs need to know where in the world you and Carmen Sandiego™ are. So, click the Time Zone tab and you’ll see a world map (see Figure 5.12).


Figure 5.12  Check and set your time zone if necessary. Some programs use this information to help you schedule mail or fax transmissions to other time zones. For laptop users who travel, this can be a great boon. Just point to your new location and click: The computer’s time is automatically adjusted.

TIP:  You can also adjust the date and time using the date and time commands from the command prompt. Another method is to use the net time command to synchronize your system time and date with a specific server on your network.

Customizing Your Screen Display

The Display applet packs a wallop under its hood. For starters, it incorporates what, in Windows 3.x, were the separate Color and Desktop Control Panel applets for prettying up the general look of the Windows screen. Then, in addition, the Display applet includes the means for changing your screen driver and resolution—functions heretofore (in Windows 3.x) available only from the Setup program. If getting at all these areas for display-tweaking from different venues annoyed you, suffer no more. Microsoft has incorporated all display-related adjustments into the unified Display applet. If you are among the blessed, you even have the option of changing screen resolution on the fly.

TIP:  The Display icon is accessible either from the Control Panel or from the Desktop. Right-click an empty area of the Desktop and choose Properties.

Here are the functional and cosmetic adjustments you can make to your Windows 2000 display from this applet:

•  Set the background and wallpaper for the Desktop.
•  Set the screen saver and energy conservation.
•  Set the color scheme and fonts for Windows elements.
•  Set the display device driver and adjust resolution, color depth, and font size.
•  Adjust the display properties of multiple monitors if the hardware provides this function.
