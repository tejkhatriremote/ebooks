To change the font of elements

1.  On the Appearance tab, simply click the element whose font you want to change. For example, if you want to change the Active Window Title, you would click the Active Window Title in the picture on the Appearance tab.
2.  In the lowest line of the dialog box, the current font for that element appears. Just open the drop-down list and choose another font if you want. You may also change the size, color, and style (bold or italic) of the font for that element.

3.  Be sure to save the scheme if you want to keep it.

Finally, consider that many screen elements—such as the borders of windows—have a constant predetermined size. However, you might want to change these settings. If you have trouble grabbing the borders of windows, for example, you might want to make them larger. If you want icons on your Desktop and in folders to line up closer or farther apart, you can do that, too. To change the size of screen elements

1.  Simply open the list and choose the item whose size you want to adjust. Some of the items are not represented in the upper section of the dialog box. They’re things that appear in other parts of Windows 2000, such as vertical icon spacing or selected items. You’ll have to experiment a bit to see the effects of these items.
2.  Click the up or down Size buttons to adjust, as shown in the following illustration.

3.  Click the Apply button to check out the effects of the changes. You might want to switch to another application via the Taskbar to see how things look.
4.  If you don’t like the effects of the changes you’ve made, you can either return to the Appearance tab of Display Properties and click Cancel or, because the screen metrics are recorded on each color scheme, you can select another color scheme.

Creating Your Own Colors

If you don’t like the colors that are available, you can create your own. There are 16 slots at the bottom of the larger color palette for storing colors you set using another fancy dialog box called the Color Refiner. Here’s how you get to the screen where you create your own colors:

1.  Click the Color button and then choose Other. This opens the enlarged color-selection box.
2.  In that box, click Define Custom Colors. Now the Color dialog box appears (see Figure 5.18).


Figure 5.18  The custom color selector lets you create new colors.

There are two cursors that you work with here. One is the luminosity bar and the other is the color refiner cursor. To make a long story short, you simply drag these around one at a time until the color in the box at the lower left is the shade you want. As you do, the numbers in the boxes below the color refiner change. Luminosity is the amount of brightness in the color. Hue is the actual shade or color. All colors are composed of red, green, and blue. Saturation is the degree of purity of the color; it is decreased by adding gray to the color and increased by subtracting it. You can also type the numbers or click the arrows next to the numbers if you want, but it’s easier to use the cursors. When you like the color, click Add to Custom Colors to add the new color to the palette.

You can switch between a solid color and a color made up of dots of several colors. Solid colors look less grainy on your screen but give you fewer choices. The Color Solid box shows the difference between the two. If you click this box before adding the color to the palette, the solid color closest to the color you chose will be added instead of the grainier composite color.

After a color is added to the palette, you can modify it. Click the color, move the cursors around, and then click Add to Custom Colors again. Click the Close button to close the dialog box. Then continue to assign colors with the palette. When you are content with the color assignments, click OK. After toying around, if you decide that you don’t want to implement the color changes, just click Cancel.

Changing the Web Settings

This tab is new to the Display settings in Windows 2000. It refers to the ability to add Web content to your Active Desktop environment. For those of you who work with Web content, this gives you the added capability of customizing the Desktop with animations, graphics, and even live content from the Internet. The Web tab of the Display Properties is shown in Figure 5.19.


Figure 5.19  Display Properties lets you set Web content for your background and control its behavior.

The main option to set on this tab is to decide whether Web content will be used on the Active Desktop. To enable this feature, simply check the Show Web Content on my Active Desktop check box.

The window at the bottom of the dialog shows the available Web pages that you can display on your Desktop. The buttons to the right side enable you to add a New Web page to the list, to Delete a page, and to Edit the properties of the selected page.

When you click the New button, you are presented with the opportunity to browse for the Web content that you want to add. This can be anything you can display on a normal Web page, or an entire Web page.

The Properties button lets you make Web pages available offline. If you select this option, the page will be cached locally for display when you aren’t connected to the Internet. When you do connect, you can update the offline page by selecting Synchronize from the Tools menu of Internet Explorer.
