Accessibility Mouse Settings

If you can’t easily control your mouse or trackball motion, or you simply don’t like using a mouse, this setting is for you. Of course, you can invoke most commands that apply to dialog boxes and menus throughout Windows and Windows programs using the Alt key in conjunction with the command’s underlined letter. Still, some programs, such as those that work with graphics, require you to use a mouse. This Accessibility option turns your arrow keys into mouse-pointer control keys. You still need to use the mouse’s clicker buttons to left- or right-click things, though.

TIP:  This is a great feature for laptop users who are on the road and forget their mouse. If you need to use a graphics program or other programs requiring more than simple command choices and text entry, use the Accessibility mouse tab to turn your arrow keys into mouse-pointer keys.

Here’s how to make or change the Accessibility Mouse settings:

1.  Click the Mouse tab in the Accessibility Options dialog box. You’ll see the dialog box displayed in Figure 5.6.


Figure 5.6  Accessibility mouse settings

2.  Turn on the option if you want to use the arrow keys in place of the mouse. You’ll probably want to adjust the speed settings for the arrow keys, so the pointer moves at a rate that works for you. The Settings button brings up the box you see in Figure 5.7. Note that you can also set a shortcut key sequence to activate MouseKeys.


Figure 5.7  Additional Accessibility mouse settings

3.  Play with the settings until you like them. The Top Speed and Acceleration settings are the most important. And note that you have to set them (click OK, then click Apply in the Mouse dialog box) before you can experience the effect of your changes. Then go back and adjust your settings if necessary. Notice that one setting lets you change the tracking speed on the fly while using a program; to slow the pointer’s motion, hold down the Shift key or, to speed it up, hold down the Ctrl key.
4.  Click the Apply button if you want to make more settings from the other tab pages or click OK to return to the Control Panel.

TIP:  The pointer keys that are used for mouse control are the ones on a standard desktop computer keyboard’s number pad. These keys have two modes—Num Lock on and Num Lock off. These keys also usually have both an arrow and a number on them; for example, the 4 key also has a left arrow symbol on it. Most laptops don’t have such keys because of size constraints. However, many laptops have a special arrangement that emulates these keys to provide a 10-key numeric keypad (and arrows when Num Lock is off).

Other Accessibility Settings

The last tab in the Accessibility applet is called General (see Figure 5.8).


Figure 5.8  The last of the Accessibility settings tabs

The tab is divided into three sections pertaining to

•  Turning on and off the Accessibility functions
•  Whether you are alerted (by sound) when a modifier key is pressed
•  Acceptance of alternative input devices through the serial (COM1 through COM4) ports on your computer

Adding and Removing Programs

The topic of adding and removing programs was first discussed in Chapters 2 and 3 in the context of running new programs and organizing your programs and documents. The Add/Remove Programs applet has two functions:

•  Installing and uninstalling programs that comply with Windows 2000’s API for these tasks. The API ensures that a program’s filenames and locations are recorded in a database, enabling them to be reliably erased without adversely affecting the operation of Windows 2000.
•  Installing and removing specific portions of Windows 2000 itself, such as Microsoft Internet Information Server.

Installing New Programs

The applet’s Install/Uninstall tab is for installing and removing programs. To install programs, take the following steps:

1.  Run Control Panel, then the Add/Remove Programs applet. You’ll see the box shown in Figure 5.9. Click the Add New Programs button to start the install process.


Figure 5.9  The wizard for adding and removing software can be reached from the Add/Remove Programs applet. Only programs that support the uninstall feature are listed here. Other applications may have separate uninstall programs.

2.  Click the CD or Floppy button. Now a new box appears, telling you to insert a floppy disk or CD-ROM in the appropriate drive and click Next. Assuming an appropriate program is found (it must be called install or setup and have a .bat, .pif, .com, or .exe extension), it’ll be displayed as you see in Figure 5.10.


Figure 5.10  The wizard looks for a likely installation program on your CD-ROM or floppy and displays the first one it finds.

3.  Click the Finish button to complete the task. The new software’s installation or setup procedure now runs. Installation instructions vary depending on the program. If your program’s setup routine isn’t compatible with the applet, you are advised of this. After installation, the new program appears in the list of removable programs only if it’s compatible with Windows 2000’s install/remove scheme.
