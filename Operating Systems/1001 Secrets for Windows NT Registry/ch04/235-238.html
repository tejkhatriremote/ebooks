This value sets the sound played when you start to drag something in an Office 97 application. For example, when you select text in a Word document and drag it someplace, you’ll hear this sound. You must provide a fully qualified path and file name to the wave file.

IV-103 Hive: HKEY_CURRENT_USER
Key: AppEvents\EventLabels\Office97-Drop
Value Name: <No Name>
Data Type: REG_SZ
Value: Drop
Key: AppEvents\Schemes\Apps\Office97\Office97-Drop
Value Name: .Default
Data Type: REG_SZ
Value: C:\WINNT\Media\Office97\Drop.wav
Value Name: .Current
Data Type: REG_SZ
Value: C:\WINNT\Media\Office97\Drop.wav
