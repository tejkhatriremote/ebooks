1. What is Windows Keylogger?

Windows Keylogger allows you to tap and protocol the Keyboard of a Windows PC System from the background.

2. How to use Windows Keylogger

At first start 'keylogger.exe'. This is the configuration tool for the real Keylogger. Determine a logfile and go to the options page . There you can activate the emailer and specify all it's necessary information like the smtp host and the file size limit for the logfile. When the logfile reaches this size the keylogger tries to send it via email. If there's no internet connection yet the keylogger saves the logfile temporarily. Also, you can activate a password protection for the configuration tool. That means only users with this password are able to start this config tool and to make any changes in the configuration.

Then go to the scheduler page if you want the Keylogger to log only at specific times or if you want to enable the self destruct option. That means that after a specific time the keylogger will uninstall itself without any messages or interaction with the user. Please note that this requires a Windows Reboot so if you select a self destruct date of, let's say, 20. July the last file (and with it the last trace of the keylogger) is deleted with the next Windows restart which may be the 22. July or even later.

On the registration/installation page you can register Windows Keylogger (more on this topic further on this page) or you can deinstall Windows Keylogger completely.

5. Was ist der Windows Keylogger?

Der Windows Keylogger ermöglicht es, das Keyboard eines Windows PC unsichtbar zu überwachen und zu protokollieren.

6. Windows Keylogger benutzen

Als erstes starten Sie 'keylogger.exe'. Das ist das Konfigurationstool für den eigentlichen Keylogger. Bestimmen Sie ein logfile und gehen Sie zur options page . Dort können Sie den emailer aktivieren und alle dafür notwendigen Informationen wie z.B. den smtp host eingeben und das file size limit für das logfile bestimmen. Wenn das logfile diese Dateigröße erreicht hat, versucht der Keylogger, es per email zu versenden. Wenn noch keine Internetverbindung vorhanden ist, speichert der Keylogger das logfile temporär und legt ein neues lofile an.
Außerdem können Sie den Paßwortschutz für das Konfigurationstool aktivieren. Dadurch sind nur User mit dem entsprechenden Paßwort in der Lage, daß Konfigurationstool zu starten und Änderungen in der Konfiguration vorzunehmen.

Gehen Sie dann zur scheduler page , wenn Sie möchten, daß der Keylogger nur zu bestimmten Zeiten arbeiten soll oder wenn Sie die self destruct option aktivieren möchten. Das bedeutet, daß der Keylogger sich nach einer bestimmten Zeit automatisch und ohne Interaktion mit dem User deinstalliert. Bitte beachten Sie, daß dies einen Windows Neustart benötigt. Wenn Sie also ein self-destruct-Datum für den 20. Juli festlegen, wird die letzte Datei (und damit die letzte Spur des Keylogger) erst beim nächsten Windows Start gelöscht, der unter Umständen durchaus einige Tage später erfolgen kann.

Auf der registration/installation page können Sie Windows Keylogger registrieren (mehr zur Registrierung erfahren Sie weiter unten) oder Sie können Windows Keylogger deinstallieren.
