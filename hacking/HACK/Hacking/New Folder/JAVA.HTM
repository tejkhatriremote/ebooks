What is Java?

Java is a portable object-oriented language developed by Sun Microsystems that is used mostly for Web programming, although it is a full-featured language that can be used to develop large applications. (There is a Web browser written entirely in Java called HotJava.)

Here are some of the features of Java that make it an attractive programming environment:

  • Java programs will work across platforms because they are run on the "Java Virtual Machine" (JVM) instead of the computer's native architecture. The JVM is a sort of simulated computer-within-a-computer.
  • Many powerful features, such as Internet connectivity and Graphical User Interface tools, are built into the standard library.
  • Java is an open standard -- all the technical information about Java internals is freely available. You can obtain free programming tools and distribute your programs without paying anyone royalties.
  • Java has a simpler, less cluttered syntax than C++.
  • Java has built-in garbage collecting -- no pointers!
  • Java is object-oriented.
  • Java was designed with security in mind; there are safeguards (although not perfect ones) against rogue Java programs on the Web.
  • Java is multi-threaded -- regardless of the host operating system, a Java program can have several parts of the program running at once.
  • You can distribute Java applets even on systems that won't allow you to write CGI scripts.

Note that Java programs come in two flavors: applications and applets. Basically speaking, applications are stand-alone programs; applets are what you see when you encounter Java on the Web. Applets have much tighter security restrictions than do applications.

Note also that JavaScript is not the same thing as Java.

For more information about Java, you can consult the Java FAQ or the Java Home Page.

If you're interested in learning Java, there are several books and tutorials available for free on the Web. One can be found at:
