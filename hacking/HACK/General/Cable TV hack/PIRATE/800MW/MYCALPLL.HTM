Mycal's PLL Project.

OK here is a very preliminary design. Audio is feed into TP2. Also note that some values are undefined. I used a shared data buss to control the PLL, LCD, input switches, and serial EEPROM. This enabled me to use a small pic to run the whole show. You need a Postscript viewer to view the schematic board layout.
